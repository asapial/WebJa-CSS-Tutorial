CSS Layout Techniques

1. Single-Column Layout

A single-column layout is a simple layout often used for mobile devices or minimalist web designs. It consists of a single column of content that stretches from one side of the viewport to the other.
  • Key Points:
    • Often used for mobile-first designs.
    • Simple and easy to read.
    • Suitable for content-focused websites like blogs or news sites.
  • Example:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Single-Column Layout</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            margin: 0;
            padding: 0;
        }

        .container {
            width: 100%;
            margin: auto;
            padding: 20px;
        }

        .header, .content, .footer {
            padding: 20px;
            margin: 10px 0;
            background: #f4f4f4;
            border: 1px solid #ddd;
        }
    </style>
</head>
<body>
    <div class="container">
        <div class="header">Header</div>
        <div class="content">Content</div>
        <div class="footer">Footer</div>
    </div>
</body>
</html>

2. Multi-Column Layout

A multi-column layout uses multiple columns to display content side-by-side. This layout is suitable for desktops and larger screens.
  • Key Points:
    • Useful for presenting content side-by-side.
    • Often used in magazines, newspapers, and content-rich websites.
    • Achieved using CSS Grid or Flexbox.
  • Example:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Multi-Column Layout</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            margin: 0;
            padding: 0;
        }

        .container {
            display: flex;
            flex-wrap: wrap;
            margin: auto;
            padding: 20px;
        }

        .header, .footer {
            width: 100%;
            padding: 20px;
            background: #f4f4f4;
            border: 1px solid #ddd;
        }

        .content {
            display: flex;
            flex-wrap: wrap;
            gap: 10px;
        }

        .sidebar, .main-content {
            flex: 1;
            padding: 20px;
            background: #f4f4f4;
            border: 1px solid #ddd;
        }
    </style>
</head>
<body>
    <div class="container">
        <div class="header">Header</div>
        <div class="content">
            <div class="sidebar">Sidebar</div>
            <div class="main-content">Main Content</div>
        </div>
        <div class="footer">Footer</div>
    </div>
</body>
</html>

3. Fixed Layout

A fixed layout uses fixed-width elements that do not change size based on the viewport. This layout is consistent across different devices but may cause horizontal scrolling on smaller screens.
  • Key Points:
    • Widths are set in pixels.
    • Ensures consistent layout appearance.
    • Not responsive to different screen sizes.
  • Example:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Fixed Layout</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            margin: 0;
            padding: 0;
        }

        .container {
            width: 960px;
            margin: auto;
            padding: 20px;
        }

        .header, .content, .footer {
            padding: 20px;
            margin: 10px 0;
            background: #f4f4f4;
            border: 1px solid #ddd;
        }
    </style>
</head>
<body>
    <div class="container">
        <div class="header">Header</div>
        <div class="content">Content</div>
        <div class="footer">Footer</div>
    </div>
</body>
</html>

4. Fluid Layout

A fluid layout uses percentage-based widths to create a design that adjusts to the size of the viewport. This layout is responsive and adapts to different screen sizes.
  • Key Points:
    • Widths are set in percentages.
    • Adapts to different screen sizes.
    • Suitable for responsive designs.
  • Example:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Fluid Layout</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            margin: 0;
            padding: 0;
        }

        .container {
            width: 90%;
            margin: auto;
            padding: 20px;
        }

        .header, .content, .footer {
            padding: 20px;
            margin: 10px 0;
            background: #f4f4f4;
            border: 1px solid #ddd;
        }
    </style>
</head>
<body>
    <div class="container">
        <div class="header">Header</div>
        <div class="content">Content</div>
        <div class="footer">Footer</div>
    </div>
</body>
</html>

5. Hybrid Layout

A hybrid layout combines different layout techniques to create a flexible and responsive design. This layout often uses a combination of fixed, fluid, and flexible elements.
  • Key Points:
    • Combines multiple layout techniques.
    • Provides flexibility and responsiveness.
    • Suitable for complex web designs.
  • Example:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Hybrid Layout</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            margin: 0;
            padding: 0;
        }

        .container {
            display: flex;
            flex-wrap: wrap;
            margin: auto;
            padding: 20px;
        }

        .header, .footer {
            width: 100%;
            padding: 20px;
            background: #f4f4f4;
            border: 1px solid #ddd;
        }

        .content {
            display: flex;
            flex: 1;
            flex-wrap: wrap;
            gap: 10px;
        }

        .sidebar {
            flex: 1;
            min-width: 200px;
            padding: 20px;
            background: #f4f4f4;
            border: 1px solid #ddd;
        }

        .main-content {
            flex: 3;
            min-width: 300px;
            padding: 20px;
            background: #f4f4f4;
            border: 1px solid #ddd;
        }
    </style>
</head>
<body>
    <div class="container">
        <div class="header">Header</div>
        <div class="content">
            <div class="sidebar">Sidebar</div>
            <div class="main-content">Main Content</div>
        </div>
        <div class="footer">Footer</div>
    </div>
</body>
</html>
