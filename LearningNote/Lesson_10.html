Borders and Outlines

The border-style property specifies the style of the border.
Syntax: 
border-style: none; /* No border */
border-style: solid; /* Solid border */
border-style: dashed; /* Dashed border */
border-style: dotted; /* Dotted border */
border-style: double; /* Double border */
border-style: groove; /* 3D grooved border */
border-style: ridge; /* 3D ridged border */
border-style: inset; /* 3D inset border */
border-style: outset; /* 3D outset border */

2. Border Width

The border-width property sets the width of the border.
Syntax: 
border-width: thin; /* Thin border */
border-width: medium; /* Medium border */
border-width: thick; /* Thick border */
border-width: 5px; /* Specific width */

3. Border Color

The border-color property sets the color of the border.
Syntax: 
border-color: red; /* Color name */
border-color: #ff0000; /* Hex value */
border-color: rgb(255, 0, 0); /* RGB value */
border-color: rgba(255, 0, 0, 0.5); /* RGBA value */

4. Border Radius

The border-radius property rounds the corners of an element's border.
Syntax: 
border-radius: 10px; /* Single value for all corners */
border-radius: 10px 20px 30px 40px; /* Different values for each corner: top-left, top-right, bottom-right, bottom-left */
border-radius: 50%; /* Circular corners */

5. Outlines

The outline property sets a line outside the border edge of an element. Outlines do not take up space and can be non-rectangular.
Syntax: 
outline: 2px solid red; /* Shorthand for setting outline width, style, and color */
outline-width: 2px; /* Width of the outline */
outline-style: solid; /* Style of the outline */
outline-color: red; /* Color of the outline */
outline-offset: 5px; /* Space between the border edge and the outline */

Example with All Topics

Here's a comprehensive example demonstrating the use of border styles, widths, colors, radius, and outlines in a CSS layout: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Borders and Outlines Example</title>
    <style>
        body {
            margin: 0;
            font-family: Arial, sans-serif;
            display: flex;
            justify-content: center;
            align-items: center;
            height: 100vh;
            background: #f0f0f0;
        }

        .box {
            width: 300px;
            height: 200px;
            background: #ffffff;
            /* Border styles */
            border-style: solid;
            border-width: 5px;
            border-color: #4285f4;
            /* Border radius */
            border-radius: 20px;
            /* Outline */
            outline: 2px dashed #ea4335;
            outline-offset: 10px;
            display: flex;
            justify-content: center;
            align-items: center;
            text-align: center;
        }
    </style>
</head>
<body>
    <div class="box">
        <p>Example with Border and Outline</p>
    </div>
</body>
</html>

Explanation

Border Styles 

border-style: solid;
  • Applies a solid style to the border.

Border Width 

border-width: 5px;
  • Sets the width of the border to 5 pixels.

Border Color 

border-color: #4285f4;
  • Sets the color of the border to a specific blue hex value.

Border Radius 

border-radius: 20px;
  • Rounds the corners of the border with a radius of 20 pixels.

Outline 

outline: 2px dashed #ea4335;
outline-offset: 10px;
  • Adds a dashed outline of 2 pixels width and red color with an offset of 10 pixels from the border edge.

Visual Effect

  • The box element has a solid blue border, rounded corners, and a red dashed outline. The outline does not affect the element's size or the border's position, and it is offset from the border by 10 pixels. This setup provides a clear distinction between borders and outlines in a styled container.
