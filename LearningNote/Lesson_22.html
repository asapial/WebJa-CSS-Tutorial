Table Styling

1. Basic Table Styling

Basic table styling involves setting the borders, background, and text alignment. 
.basic-table {
    width: 100%;
    border-collapse: collapse;
}

.basic-table th, .basic-table td {
    border: 1px solid black;
    padding: 8px;
    text-align: left;
}

.basic-table th {
    background-color: #f2f2f2;
}

2. Table Borders

Table borders can be styled to have different thicknesses, colors, and styles. 
.table-borders {
    border-collapse: collapse;
}

.table-borders th, .table-borders td {
    border: 2px solid #4CAF50;
    padding: 8px;
}

.table-borders th {
    background-color: #d4edda;
}

3. Table Padding and Spacing

Padding and spacing improve the readability and appearance of the table. 
.table-padding-spacing {
    border-collapse: collapse;
}

.table-padding-spacing th, .table-padding-spacing td {
    padding: 15px;
    border: 1px solid black;
}

.table-padding-spacing th {
    background-color: #e9ecef;
}

4. Zebra Stripes

Zebra stripes enhance the readability of the table by alternating row colors. 
.zebra-stripes {
    border-collapse: collapse;
    width: 100%;
}

.zebra-stripes th, .zebra-stripes td {
    border: 1px solid #ddd;
    padding: 8px;
}

.zebra-stripes tr:nth-child(even) {
    background-color: #f2f2f2;
}

.zebra-stripes tr:hover {
    background-color: #ddd;
}

.zebra-stripes th {
    background-color: #4CAF50;
    color: white;
}

5. Responsive Tables

Responsive tables ensure the table looks good on different screen sizes. 
.responsive-table {
    width: 100%;
    border-collapse: collapse;
}

.responsive-table th, .responsive-table td {
    border: 1px solid #ddd;
    padding: 8px;
}

.responsive-table th {
    background-color: #f2f2f2;
}

.responsive-table tr {
    display: block;
    border-bottom: 2px solid #ddd;
}

@media screen and (min-width: 600px) {
    .responsive-table tr {
        display: table-row;
    }
}
