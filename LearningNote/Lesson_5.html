Flexbox Basics

1. Introduction to Flexbox

Flexbox is a layout model that allows you to design a flexible and efficient layout structure without using float or positioning. The main idea behind the flex layout is to give the container the ability to alter its items' width, height, and order to best fill the available space (or to adapt to a specified space).

2. Flex Container

A flex container is the parent element where the display property is set to flex or inline-flex. This container establishes a flex formatting context for its direct children. 
.flex-container {
    display: flex;
    border: 1px solid black;
    padding: 10px;
    margin-bottom: 20px;
}

3. Flex Items

Flex items are the direct children of a flex container. They can be manipulated using various flex properties. 
.flex-item {
    background-color: lightblue;
    padding: 20px;
    margin: 10px;
    border: 1px solid darkblue;
}

4. justify-content

The justify-content property aligns the flex items along the main axis (horizontally in a row layout, vertically in a column layout). Possible values include:
  • flex-start (default): Items are packed toward the start.
  • flex-end: Items are packed toward the end.
  • center: Items are centered along the line.
  • space-between: Items are evenly distributed; the first item is at the start, the last item is at the end.
  • space-around: Items are evenly distributed with equal space around them.
  • space-evenly: Items are evenly distributed with equal space between them. 
.justify-content-container {
    display: flex;
    justify-content: center; /* Change to see different effects */
    border: 1px solid black;
    padding: 10px;
    margin-bottom: 20px;
}

5. align-items

The align-items property aligns the flex items along the cross axis (vertically in a row layout, horizontally in a column layout). Possible values include:
  • stretch (default): Items stretch to fill the container.
  • flex-start: Items are aligned to the start of the container.
  • flex-end: Items are aligned to the end of the container.
  • center: Items are centered along the cross axis.
  • baseline: Items are aligned such that their baselines align. 
.align-items-container {
    display: flex;
    align-items: center; /* Change to see different effects */
    height: 200px; /* Set a fixed height to see the alignment effects */
    border: 1px solid black;
    padding: 10px;
    margin-bottom: 20px;
}

Example Code

Here is an example demonstrating the basics of Flexbox: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Flexbox Basics</title>
    <style>
        /* Flex container */
        .flex-container {
            display: flex;
            border: 1px solid black;
            padding: 10px;
            margin-bottom: 20px;
        }

        /* Flex items */
        .flex-item {
            background-color: lightblue;
            padding: 20px;
            margin: 10px;
            border: 1px solid darkblue;
        }

        /* Justify content container */
        .justify-content-container {
            display: flex;
            justify-content: space-between; /* Change to see different effects */
            border: 1px solid black;
            padding: 10px;
            margin-bottom: 20px;
        }

        /* Align items container */
        .align-items-container {
            display: flex;
            align-items: center; /* Change to see different effects */
            height: 200px; /* Set a fixed height to see the alignment effects */
            border: 1px solid black;
            padding: 10px;
            margin-bottom: 20px;
        }
    </style>
</head>
<body>
    <!-- Flex container example -->
    <div class="flex-container">
        <div class="flex-item">Item 1</div>
        <div class="flex-item">Item 2</div>
        <div class="flex-item">Item 3</div>
    </div>
    
    <!-- Justify content example -->
    <div class="justify-content-container">
        <div class="flex-item">Item 1</div>
        <div class="flex-item">Item 2</div>
        <div class="flex-item">Item 3</div>
    </div>
    
    <!-- Align items example -->
    <div class="align-items-container">
        <div class="flex-item">Item 1</div>
        <div class="flex-item">Item 2</div>
        <div class="flex-item">Item 3</div>
    </div>
</body>
</html>
