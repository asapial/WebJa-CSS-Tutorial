Advanced Pseudo-classes

1. :nth-child()

The :nth-child() pseudo-class selects elements based on their position in a group of siblings. It accepts a formula (an+b) to match elements at specific positions.
  • Syntax:
    :nth-child(an+b)
  • Example Explanation:
    In this example, odd-numbered list items are given a light gray background, and even-numbered list items are given a darker gray background.

2. :nth-of-type()

The :nth-of-type() pseudo-class selects elements based on their position among siblings of the same type.
  • Syntax:
    :nth-of-type(an+b)
  • Example Explanation:
    Here, the second paragraph (<p>) within a container is styled differently.

3. :first-child

The :first-child pseudo-class selects an element if it is the first child of its parent.
  • Syntax:
    :first-child
  • Example Explanation:
    The first child of each list within a container is given a specific style.

4. :last-child

The :last-child pseudo-class selects an element if it is the last child of its parent.
  • Syntax:
    :last-child
  • Example Explanation:
    The last child of each list within a container is given a specific style.

5. :not()

The :not() pseudo-class selects elements that do not match a specified selector.
  • Syntax:
    :not(selector)
  • Example Explanation:
    List items that are not the third child are given a specific style.

Example Code 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Advanced Pseudo-classes</title>
    <style>
        /* nth-child() example */
        .nth-child-example li:nth-child(odd) {
            background-color: #f0f0f0; /* Light gray for odd items */
        }

        .nth-child-example li:nth-child(even) {
            background-color: #d3d3d3; /* Darker gray for even items */
        }

        /* nth-of-type() example */
        .nth-of-type-example p:nth-of-type(2) {
            color: #3498db; /* Blue text for the second paragraph */
        }

        /* first-child example */
        .first-child-example li:first-child {
            font-weight: bold; /* Bold text for the first child */
        }

        /* last-child example */
        .last-child-example li:last-child {
            font-style: italic; /* Italic text for the last child */
        }

        /* not() example */
        .not-example li:not(:nth-child(3)) {
            text-decoration: underline; /* Underline all items except the third */
        }
    </style>
</head>
