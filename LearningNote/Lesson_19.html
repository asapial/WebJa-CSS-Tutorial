CSS Filters

1. blur()

The blur() function applies a Gaussian blur to the element. The parameter defines the radius of the blur effect.
Syntax: 
filter: blur(radius);
  • radius: The amount of blur. It can be specified in pixels (px), ems (em), or other length units.
Example: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS Filter: Blur</title>
    <style>
        .blur {
            filter: blur(5px);
        }
    </style>
</head>
<body>
    <h1>CSS Filter: Blur Example</h1>
    <img src="example.jpg" alt="Example Image" class="blur">
</body>
</html>

2. brightness()

The brightness() function adjusts the brightness of the element. The parameter specifies the brightness level.
Syntax: 
filter: brightness(value);
  • value: The brightness level. Values greater than 1 increase brightness, values between 0 and 1 decrease brightness, and 0 makes the element completely black.
Example: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS Filter: Brightness</title>
    <style>
        .brightness {
            filter: brightness(1.5);
        }
    </style>
</head>
<body>
    <h1>CSS Filter: Brightness Example</h1>
    <img src="example.jpg" alt="Example Image" class="brightness">
</body>
</html>

4. drop-shadow()

The drop-shadow() function applies a drop shadow effect to the element. The parameters specify the shadow's offset, blur radius, and color.
Syntax: 
filter: drop-shadow(offsetX offsetY blurRadius color);
  • offsetX: The horizontal offset of the shadow.
  • offsetY: The vertical offset of the shadow.
  • blurRadius: The blur radius of the shadow.
  • color: The color of the shadow.
Example: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS Filter: Drop Shadow</title>
    <style>
        .drop-shadow {
            filter: drop-shadow(10px 10px 5px rgba(0, 0, 0, 0.5));
        }
    </style>
</head>
<body>
    <h1>CSS Filter: Drop Shadow Example</h1>
    <img src="example.jpg" alt="Example Image" class="drop-shadow">
</body>
</html>

5. grayscale()

The grayscale() function converts the element to grayscale. The parameter specifies the amount of conversion.
Syntax: 
filter: grayscale(value);
  • value: The grayscale level. A value of 1 makes the element completely grayscale, while a value of 0 leaves it unchanged.
Example: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS Filter: Grayscale</title>
    <style>
        .grayscale {
            filter: grayscale(1);
        }
    </style>
</head>
<body>
    <h1>CSS Filter: Grayscale Example</h1>
    <img src="example.jpg" alt="Example Image" class="grayscale">
</body>
</html>

Example Code 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Interactive CSS Filters Example</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            margin: 20px;
        }
        .container {
            display: flex;
            flex-direction: column;
            align-items: center;
        }
        img {
            max-width: 100%;
            height: auto;
            margin-bottom: 20px;
        }
        .controls {
            display: flex;
            flex-direction: column;
            align-items: center;
        }
        .control {
            margin-bottom: 10px;
        }
        label {
            display: inline-block;
            width: 100px;
        }
        input[type="range"] {
            width: 300px;
        }
    </style>
</head>
<body>
    <div class="container">
        <h1>Interactive CSS Filters Example</h1>
        <img src="https://via.placeholder.com/400" alt="Example Image" id="image">
        <div class="controls">
            <div class="control">
                <label for="blur">Blur:</label>
                <input type="range" id="blur" min="0" max="20" step="1" value="0" oninput="updateFilter()">
            </div>
            <div class="control">
                <label for="brightness">Brightness:</label>
                <input type="range" id="brightness" min="0" max="2" step="0.1" value="1" oninput="updateFilter()">
            </div>
            <div class="control">
                <label for="contrast">Contrast:</label>
                <input type="range" id="contrast" min="0" max="3" step="0.1" value="1" oninput="updateFilter()">
            </div>
            <div class="control">
                <label for="drop-shadow-x">Shadow X:</label>
                <input type="range" id="drop-shadow-x" min="-50" max="50" step="1" value="0" oninput="updateFilter()">
            </div>
            <div class="control">
                <label for="drop-shadow-y">Shadow Y:</label>
                <input type="range" id="drop-shadow-y" min="-50" max="50" step="1" value="0" oninput="updateFilter()">
            </div>
            <div class="control">
                <label for="drop-shadow-blur">Shadow Blur:</label>
                <input type="range" id="drop-shadow-blur" min="0" max="20" step="1" value="0" oninput="updateFilter()">
            </div>
            <div class="control">
                <label for="grayscale">Grayscale:</label>
                <input type="range" id="grayscale" min="0" max="1" step="0.1" value="0" oninput="updateFilter()">
            </div>
        </div>
    </div>

    <script>
        function updateFilter() {
            const blur = document.getElementById('blur').value;
            const brightness = document.getElementById('brightness').value;
            const contrast = document.getElementById('contrast').value;
            const dropShadowX = document.getElementById('drop-shadow-x').value;
            const dropShadowY = document.getElementById('drop-shadow-y').value;
            const dropShadowBlur = document.getElementById('drop-shadow-blur').value;
            const grayscale = document.getElementById('grayscale').value;

            const image = document.getElementById('image');
            image.style.filter = `
                blur(${blur}px)
                brightness(${brightness})
                contrast(${contrast})
                drop-shadow(${dropShadowX}px ${dropShadowY}px ${dropShadowBlur}px rgba(0, 0, 0, 0.5))
                grayscale(${grayscale})
            `;
        }
    </script>
</body>
</html>
