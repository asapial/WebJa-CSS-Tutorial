Backgrounds

1. background-image

The background-image property sets one or more background images for an element.
Syntax: 
background-image: url('path/to/image.jpg');
You can also set multiple background images: 
background-image: url('image1.jpg'), url('image2.jpg');

2. background-repeat

The background-repeat property defines how background images are repeated. It can be repeated along the x-axis, y-axis, both, or not at all.
Syntax: 
background-repeat: repeat; /* Default */
background-repeat: no-repeat;
background-repeat: repeat-x;
background-repeat: repeat-y;
background-repeat: space;
background-repeat: round;

3. background-size

The background-size property specifies the size of the background images.
Syntax: 
background-size: auto; /* Default */
background-size: cover;
background-size: contain;
background-size: 50% 50%; /* width height */

4. background-position

The background-position property sets the initial position of the background images.
Syntax: 
background-position: left top; /* Default */
background-position: 50% 50%; /* x y */
background-position: center;
background-position: right bottom;

5. background-attachment

The background-attachment property sets whether a background image scrolls with the page or is fixed.
Syntax: 
background-attachment: scroll; /* Default */
background-attachment: fixed;
background-attachment: local;

Example with All Topics

Here's a comprehensive example demonstrating the use of all these background properties in a CSS layout: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Background Properties Example</title>
    <style>
        body {
            margin: 0;
            font-family: Arial, sans-serif;
        }

        .background-example {
            width: 100%;
            height: 100vh;
            /* Set a background image */
            background-image: url('https://via.placeholder.com/800x600');
            /* Do not repeat the background image */
            background-repeat: no-repeat;
            /* Cover the entire element */
            background-size: cover;
            /* Position the background image */
            background-position: center;
            /* Fix the background image */
            background-attachment: fixed;
        }

        .content {
            height: 200vh; /* Just to create scrolling effect */
            background: rgba(255, 255, 255, 0.8); /* Semi-transparent background for content */
            padding: 20px;
        }
    </style>
</head>
<body>
    <div class="background-example">
        <div class="content">
            <h1>Background Properties Example</h1>
            <p>This is an example demonstrating the use of various background properties in CSS. Scroll to see the effect of the fixed background.</p>
        </div>
    </div>
</body>
</html>

Explanation

1. background-image 

background-image: url('https://via.placeholder.com/800x600');
  • Sets the background image to an example image from a placeholder URL.

2. background-repeat 

background-repeat: no-repeat;
  • Ensures the background image does not repeat.

3. background-size 

background-size: cover;
  • Scales the background image to cover the entire element, while maintaining its aspect ratio.

4. background-position 

background-position: center;
  • Positions the background image at the center of the element.

5. background-attachment 

background-attachment: fixed;
  • Fixes the background image so it does not scroll with the content.
