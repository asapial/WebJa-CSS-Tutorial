Positioning

1. Static Positioning

Static positioning is the default positioning for HTML elements. Elements are positioned according to the normal flow of the document, meaning they are placed one after the other in the order they appear in the HTML. 
/* No need for special CSS for static positioning */
div {
    background-color: lightblue;
    padding: 10px;
}

2. Relative Positioning

Relative positioning allows you to position an element relative to its normal position. The space for the element is still reserved in the normal document flow. 
.relative {
    position: relative;
    top: 20px; /* Moves the element 20px down from its normal position */
    left: 30px; /* Moves the element 30px to the right from its normal position */
    background-color: lightgreen;
    padding: 10px;
}

3. Absolute Positioning

Absolute positioning removes the element from the normal document flow and positions it relative to its nearest positioned ancestor (an ancestor element that has a position other than static). If no such ancestor exists, it positions the element relative to the initial containing block (usually the <html> element). 
.absolute {
    position: absolute;
    top: 50px; /* Moves the element 50px down from the top of the nearest positioned ancestor */
    left: 100px; /* Moves the element 100px to the right from the left of the nearest positioned ancestor */
    background-color: lightcoral;
    padding: 10px;
}

4. Fixed Positioning

Fixed positioning removes the element from the normal document flow and positions it relative to the viewport, meaning it stays in the same place even when the page is scrolled. 
.fixed {
    position: fixed;
    top: 0; /* Sticks the element to the top of the viewport */
    left: 0; /* Sticks the element to the left of the viewport */
    background-color: lightsalmon;
    padding: 10px;
    width: 100%; /* Stretches the element across the viewport width */
}

5. Sticky Positioning

Sticky positioning is a hybrid of relative and fixed positioning. An element with position: sticky is treated as relative until it crosses a specified threshold (like top: 10px), at which point it is treated as fixed. 
.sticky {
    position: sticky;
    top: 10px; /* The element becomes fixed after scrolling past 10px from the top */
    background-color: lightyellow;
    padding: 10px;
}

Example Code

Here is an example demonstrating all these positioning concepts: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Positioning Example</title>
    <style>
        /* Container class to style individual boxes */
        .container {
            border: 1px solid black;
            padding: 10px;
            margin-bottom: 20px;
        }
        /* Static positioning (default) */
        .static {
            background-color: lightblue;
            padding: 10px;
        }
        /* Relative positioning: shifted from its normal position */
        .relative {
            position: relative;
            top: 20px; /* Moves 20px down from its normal position */
            left: 30px; /* Moves 30px to the right from its normal position */
            background-color: lightgreen;
            padding: 10px;
        }
        /* Container for the absolutely positioned element */
        .absolute-container {
            position: relative; /* This makes it the nearest positioned ancestor */
            height: 200px;
            border: 1px solid red;
            margin-bottom: 20px;
        }
        /* Absolute positioning: positioned relative to the nearest positioned ancestor */
        .absolute {
            position: absolute;
            top: 50px; /* 50px down from the top of the absolute-container */
            left: 100px; /* 100px from the left of the absolute-container */
            background-color: lightcoral;
            padding: 10px;
        }
        /* Fixed positioning: positioned relative to the viewport */
        .fixed {
            position: fixed;
            top: 0; /* Sticks to the top of the viewport */
            left: 0; /* Sticks to the left of the viewport */
            background-color: lightsalmon;
            padding: 10px;
            width: 100%; /* Stretches across the width of the viewport */
            z-index: 1000; /* Ensures it is on top of other elements */
        }
        /* Sticky positioning: switches between relative and fixed based on scroll */
        .sticky {
            position: sticky;
            top: 10px; /* Sticks to 10px from the top once scrolled past */
            background-color: lightyellow;
            padding: 10px;
        }
    </style>
</head>
<body>
    <!-- Static positioning example -->
    <div class="container static">
        This is a static positioned box (default).
    </div>
    
    <!-- Relative positioning example -->
    <div class="container relative">
        This is a relatively positioned box.
    </div>
    
    <!-- Container for absolute positioning example -->
    <div class="absolute-container">
        This is the container for an absolutely positioned box.
        <!-- Absolute positioning example -->
        <div class="absolute">
            This is an absolutely positioned box.
        </div>
    </div>
    
    <!-- Fixed positioning example -->
    <div class="fixed">
        This is a fixed positioned box (scroll to see the effect).
    </div>
    
    <!-- Spacer to enable scrolling -->
    <div style="height: 1500px;">
        <!-- Empty space to allow scrolling -->
    </div>
    
    <!-- Sticky positioning example -->
    <div class="container sticky">
        This is a sticky positioned box (scroll to see the effect).
    </div>
</body>
</html>
