Basics of CSS

1. Introduction to CSS

Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML or XML. CSS defines how elements should be rendered on screen, on paper, in speech, or on other media. By separating the content (HTML) from the presentation (CSS), you can improve the maintainability, accessibility, and performance of web pages.

2. CSS Syntax and Selectors

CSS Syntax:
CSS rules are composed of a selector and a declaration block.
  • Selector: Indicates which HTML element(s) the rule applies to.
  • Declaration Block: Contains one or more declarations separated by semicolons. Each declaration includes a property and a value, separated by a colon. 
selector {
  property: value;
  property: value;
}
Common Selectors:
  • Element Selector: Selects all elements of a specified type. 
    p {
  color: blue;
}
  • Class Selector: Selects elements with a specific class attribute. Prefixed with a dot (.). 
    .classname {
  color: red;
}
  • ID Selector: Selects an element with a specific id attribute. Prefixed with a hash (#). 
    #idname {
  color: green;
}
  • Attribute Selector: Selects elements with a specified attribute. 
    [type="text"] {
  border: 1px solid black;
}

3. Applying CSS to HTML (inline, internal, external)

  • Inline CSS: Uses the style attribute directly within HTML elements. 
    <p style="color: blue;">This is a blue paragraph.</p>
  • Internal CSS: Uses the <style> tag within the <head> section of an HTML document. 
    <head>
  <style>
    p {
      color: blue;
    }
  </style>
</head>
  • External CSS: Links to an external CSS file using the <link> tag within the <head> section. 
    <head>
  <link rel="stylesheet" href="styles.css">
</head>

4. CSS Comments

CSS comments are used to explain your code and are ignored by browsers. Comments are enclosed within /* */. 
/* This is a comment */
p {
  color: blue; /* This comment is inside a declaration */
}

5. Basic CSS Properties (color, background-color)

  • color: Sets the color of the text. 
    p {
  color: blue;
}
  • background-color: Sets the background color of an element. 
    p {
  background-color: yellow;
}
