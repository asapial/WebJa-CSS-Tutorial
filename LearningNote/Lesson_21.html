Forms and Input Styling

1. Styling Text Inputs

CSS for Text Inputs: 
input[type="text"], input[type="email"], input[type="password"] {
    padding: 10px;
    border: 2px solid #ccc;
    border-radius: 5px;
    font-size: 16px;
    width: 100%;
    box-sizing: border-box;
    margin-bottom: 10px;
}

input[type="text"]:focus, input[type="email"]:focus, input[type="password"]:focus {
    border-color: #007BFF;
    outline: none;
}

2. Styling Buttons

CSS for Buttons: 
button {
    background-color: #007BFF;
    color: white;
    padding: 10px 20px;
    border: none;
    border-radius: 5px;
    cursor: pointer;
    font-size: 16px;
    transition: background-color 0.3s ease;
    margin-top: 10px;
}

button:hover {
    background-color: #0056b3;
}

3. Styling Checkboxes

CSS for Checkboxes: 
input[type="checkbox"] {
    width: 20px;
    height: 20px;
    cursor: pointer;
    accent-color: #007BFF;
    margin-right: 10px;
}

input[type="checkbox"]:checked {
    background-color: #007BFF;
    border-color: #007BFF;
}

4. Styling Radio Buttons

CSS for Radio Buttons: 
input[type="radio"] {
    width: 20px;
    height: 20px;
    cursor: pointer;
    accent-color: #007BFF;
    margin-right: 10px;
}

input[type="radio"]:checked {
    background-color: #007BFF;
    border-color: #007BFF;
}

5. Styling Select Menus

CSS for Select Menus: 
select {
    padding: 10px;
    border: 2px solid #ccc;
    border-radius: 5px;
    font-size: 16px;
    width: 100%;
    box-sizing: border-box;
    appearance: none;
    background-color: white;
    background-image: url('data:image/svg+xml;utf8,<svg fill="%23000000" height="24" viewBox="0 0 24 24" width="24" xmlns="http://www.w3.org/2000/svg"><path d="M7 10l5 5 5-5z"/></svg>');
    background-repeat: no-repeat;
    background-position: right 10px center;
    background-size: 12px;
    margin-bottom: 10px;
}

select:focus {
    border-color: #007BFF;
    outline: none;
}
