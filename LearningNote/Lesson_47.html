CSS for Web Components

1. Shadow DOM

Description:
The Shadow DOM is a key feature of Web Components that encapsulates the internal DOM structure of a custom element. It allows for style and markup encapsulation, preventing styles from leaking out or being affected by external styles.
Example: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Shadow DOM Example</title>
</head>
<body>
    <my-element></my-element>

    <script>
        class MyElement extends HTMLElement {
            constructor() {
                super();
                const shadow = this.attachShadow({ mode: 'open' });
                shadow.innerHTML = `
                    <style>
                        .container {
                            padding: 20px;
                            background-color: lightblue;
                            border: 1px solid #ccc;
                        }
                    </style>
                    <div class="container">
                        <p>This is a shadow DOM element</p>
                    </div>
                `;
            }
        }

        customElements.define('my-element', MyElement);
    </script>
</body>
</html>

2. Styling Shadow DOM Elements

Description:
Styling elements within the Shadow DOM is done using the <style> tag within the shadow root. These styles are scoped to the Shadow DOM and do not affect or get affected by the main document styles.
Example: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Styling Shadow DOM Elements</title>
</head>
<body>
    <my-styled-element></my-styled-element>

    <script>
        class MyStyledElement extends HTMLElement {
            constructor() {
                super();
                const shadow = this.attachShadow({ mode: 'open' });
                shadow.innerHTML = `
                    <style>
                        .box {
                            padding: 20px;
                            background-color: lightcoral;
                            color: white;
                            border-radius: 5px;
                            text-align: center;
                        }
                    </style>
                    <div class="box">
                        <p>Styled Shadow DOM Element</p>
                    </div>
                `;
            }
        }

        customElements.define('my-styled-element', MyStyledElement);
    </script>
</body>
</html>

3. CSS Custom Properties in Web Components

Description:
CSS custom properties (variables) can be used within Web Components to allow for easier theming and customization. They can be defined and accessed within the Shadow DOM.
Example: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS Custom Properties in Web Components</title>
    <style>
        :root {
            --box-bg-color: lightgreen;
            --box-padding: 20px;
        }
    </style>
</head>
<body>
    <my-custom-element></my-custom-element>

    <script>
        class MyCustomElement extends HTMLElement {
            constructor() {
                super();
                const shadow = this.attachShadow({ mode: 'open' });
                shadow.innerHTML = `
                    <style>
                        .box {
                            padding: var(--box-padding);
                            background-color: var(--box-bg-color);
                            color: black;
                            border: 1px solid #ccc;
                        }
                    </style>
                    <div class="box">
                        <p>Custom Properties in Shadow DOM</p>
                    </div>
                `;
            }
        }

        customElements.define('my-custom-element', MyCustomElement);
    </script>
</body>
</html>

4. Scoped CSS

Description:
Scoped CSS ensures that styles defined within a Web Component do not leak out to the main document and vice versa. This is achieved using the Shadow DOM.
Example: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Scoped CSS Example</title>
    <style>
        .main {
            padding: 20px;
            background-color: lightyellow;
        }
    </style>
</head>
<body>
    <div class="main">
        <p>Main Document Style</p>
    </div>
    <my-scoped-element></my-scoped-element>

    <script>
        class MyScopedElement extends HTMLElement {
            constructor() {
                super();
                const shadow = this.attachShadow({ mode: 'open' });
                shadow.innerHTML = `
                    <style>
                        .scoped {
                            padding: 20px;
                            background-color: lightblue;
                        }
                    </style>
                    <div class="scoped">
                        <p>Scoped CSS in Shadow DOM</p>
                    </div>
                `;
            }
        }

        customElements.define('my-scoped-element', MyScopedElement);
    </script>
</body>
</html>

5. Using CSS with LitElement

Description:
LitElement is a base class for creating Web Components with lit-html and is designed to make it easier to use Shadow DOM, scoped CSS, and reactive properties.
Example: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Using CSS with LitElement</title>
    <script type="module">
        import { LitElement, html, css } from 'lit';

        class MyLitElement extends LitElement {
            static styles = css`
                .box {
                    padding: 20px;
                    background-color: lightblue;
                    color: black;
                    border: 1px solid #ccc;
                }
            `;

            render() {
                return html`
                    <div class="box">
                        <p>LitElement with Scoped CSS</p>
                    </div>
                `;
            }
        }

        customElements.define('my-lit-element', MyLitElement);
    </script>
</head>
<body>
    <my-lit-element></my-lit-element>
</body>
</html>
