Advanced Grid Layout

1. grid-area

The grid-area property allows you to assign a name to a grid item and position it within the grid. It can be used in two main ways:
  • Assigning a name to a grid item to be referenced in grid-template-areas.
  • Using a shorthand syntax to specify the start and end lines for both rows and columns.
Syntax: 
/* Using a name */
grid-area: header;

/* Using line numbers (row-start / column-start / row-end / column-end) */
grid-area: 1 / 1 / 2 / 4;

2. grid-template-areas

The grid-template-areas property defines a grid template by specifying the layout of named grid areas. This is done by creating a string of space-separated names enclosed in quotes, with each string representing a row.
Syntax: 
grid-template-areas:
  "header header header"
  "sidebar main main"
  "footer footer footer";

3. justify-items

The justify-items property aligns grid items horizontally within their grid area. Possible values are start, end, center, and stretch (default).
Syntax: 
justify-items: center;

4. align-items

The align-items property aligns grid items vertically within their grid area. Possible values are start, end, center, and stretch (default).
Syntax: 
align-items: start;

5. grid-auto-flow

The grid-auto-flow property controls how auto-placed items are inserted into the grid. Possible values are row, column, row dense, and column dense.
Syntax: 
grid-auto-flow: row;

Example with All Topics

Here is a comprehensive example demonstrating the use of all these properties in a CSS Grid layout: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Advanced Grid Layout Example</title>
    <style>
        body {
            margin: 0;
            font-family: Arial, sans-serif;
        }

        .container {
            display: grid;
            /* Define the grid template areas */
            grid-template-areas:
                "header header header"
                "sidebar main main"
                "footer footer footer";
            /* Set the column and row sizes */
            grid-template-columns: 1fr 2fr 1fr;
            grid-template-rows: auto 1fr auto;
            /* Align items horizontally within their grid area */
            justify-items: center;
            /* Align items vertically within their grid area */
            align-items: center;
            /* Control the placement of auto-placed items */
            grid-auto-flow: row;
            gap: 10px; /* Optional: Add some space between grid items */
        }

        .header {
            grid-area: header;
            background: #f8b400;
        }

        .sidebar {
            grid-area: sidebar;
            background: #34a853;
        }

        .main {
            grid-area: main;
            background: #4285f4;
        }

        .footer {
            grid-area: footer;
            background: #ea4335;
        }

        .item {
            padding: 20px;
            color: white;
            font-size: 1.5em;
            text-align: center;
        }

        .extra1, .extra2 {
            background: #777;
        }

        /* Use grid-area shorthand to place extra items */
        .extra1 {
            grid-area: 2 / 3 / 3 / 4; /* Placed in row 2, column 3 */
        }

        .extra2 {
            grid-area: 3 / 1 / 4 / 2; /* Placed in row 3, column 1 */
        }
    </style>
</head>
<body>
    <div class="container">
        <div class="header item">Header</div>
        <div class="sidebar item">Sidebar</div>
        <div class="main item">Main Content</div>
        <div class="footer item">Footer</div>
        <div class="extra1 item">Extra Item 1</div>
        <div class="extra2 item">Extra Item 2</div>
    </div>
</body>
</html>

Explanation

Grid Container (.container)

  • display: grid;: Turns the container into a grid layout.
  • grid-template-areas: Defines the grid template with named areas (header, sidebar, main, footer). Each string in the property represents a row in the grid.
  • grid-template-columns: 1fr 2fr 1fr;: Defines three columns. The first and third columns each take up 1 fraction unit (1fr), and the middle column takes up 2 fraction units (2fr).
  • grid-template-rows: auto 1fr auto;: Defines three rows. The first and third rows are auto-sized, and the middle row takes up 1 fraction unit (1fr).
  • justify-items: center;: Centers grid items horizontally within their grid areas.
  • align-items: center;: Centers grid items vertically within their grid areas.
  • grid-auto-flow: row;: Specifies that auto-placed items should be placed in rows.
  • gap: 10px;: Adds a 10px gap between grid items.

Grid Items

  • .header, .sidebar, .main, .footer: These classes use grid-area to place items in the grid's named areas.
    • grid-area: header;: Places the item in the area named header.
    • grid-area: sidebar;: Places the item in the area named sidebar.
    • grid-area: main;: Places the item in the area named main.
    • grid-area: footer;: Places the item in the area named footer.
  • .item: Applies common styling to all grid items (padding, color, font size, text alignment).
  • .extra1, .extra2: These additional items are positioned using the grid-area shorthand to specify their start and end lines:
    • grid-area: 2 / 3 / 3 / 4;: Places extra1 in the second row, third column, spanning 1 row and 1 column.
    • grid-area: 3 / 1 / 4 / 2;: Places extra2 in the third row, first column, spanning 1 row and 1 column.
