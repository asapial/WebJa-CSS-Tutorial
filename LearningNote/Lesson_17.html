CSS Functions

1. calc()

The calc() function allows you to perform calculations to determine CSS property values.
  • Syntax:
    width: calc(100% - 50px);
  • Example:
    <div class="calc-box">This box uses calc()</div>

    .calc-box {
    width: calc(100% - 50px);
    padding: 20px;
    background-color: lightcoral;
}

2. var()

The var() function is used to access the value of a CSS variable (custom property).
  • Syntax:
    color: var(--primary-color);
  • Example:
    <div class="var-box">This box uses var()</div>

    :root {
    --primary-color: #3498db;
}

.var-box {
    color: var(--primary-color);
    padding: 20px;
    background-color: #f0f0f0;
}

3. url()

The url() function is used to include a file, such as an image, within a CSS property.
  • Syntax:
    background-image: url('path/to/image.jpg');
  • Example:
    <div class="url-box">This box uses url()</div>

    .url-box {
    width: 300px;
    height: 200px;
    background-image: url('https://via.placeholder.com/300x200');
    background-size: cover;
    padding: 20px;
    color: white;
}

4. rgba()

The rgba() function defines colors using red, green, blue, and alpha (opacity) values.
  • Syntax:
    background-color: rgba(255, 99, 71, 0.7);
  • Example:
    <div class="rgba-box">This box uses rgba()</div>

    .rgba-box {
    padding: 20px;
    background-color: rgba(255, 99, 71, 0.7); /* Semi-transparent tomato color */
}

5. hsla()

The hsla() function defines colors using hue, saturation, lightness, and alpha (opacity) values.
  • Syntax:
    background-color: hsla(120, 100%, 50%, 0.3);
  • Example:
    <div class="hsla-box">This box uses hsla()</div>

    .hsla-box {
    padding: 20px;
    background-color: hsla(120, 100%, 50%, 0.3); /* Semi-transparent green */
}
