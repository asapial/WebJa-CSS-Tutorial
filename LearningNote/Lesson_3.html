Box Model

1. Understanding the Box Model

The CSS box model describes the rectangular boxes generated for elements in the document tree. It consists of the following parts:
  • Content: The inner part where text and images appear.
  • Padding: Clears an area around the content; its size is affected by the background color of the box.
  • Border: A border that goes around the padding (and content).
  • Margin: Clears an area outside the border; its size is not affected by the background color of the box.

2. Content, Padding, Border, Margin

Each part of the box model can be manipulated using CSS properties:
  • Content: The width and height properties affect the content area.
  • Padding: Controlled with padding, padding-top, padding-right, padding-bottom, and padding-left.
  • Border: Controlled with border, border-width, border-style, and border-color.
  • Margin: Controlled with margin, margin-top, margin-right, margin-bottom, and margin-left.

3. Width and Height

The width and height properties specify the size of the content area. The total size of an element, including padding, border, and margin, is calculated as: 
Total width = width + left padding + right padding + left border + right border + left margin + right margin
Total height = height + top padding + bottom padding + top border + bottom border + top margin + bottom margin

4. Box Sizing

The box-sizing property allows you to change the box model used by the browser. By default, the width and height properties include only the content area. The box-sizing property can be set to content-box or border-box.
  • content-box (default): The width and height properties include only the content. Padding, border, and margin are added outside of the box.
  • border-box: The width and height properties include the content, padding, and border. The margin is added outside of the box.

5. Display Property

The display property specifies how an element is displayed. Common values include:
  • block: The element is displayed as a block-level element.
  • inline: The element is displayed as an inline-level element.
  • inline-block: The element is displayed as an inline-level block container.
  • none: The element is not displayed.

Example Code

Here is a comprehensive example that demonstrates all these concepts: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Box Model Example</title>
    <style>
        .box {
            width: 200px;
            height: 100px;
            padding: 20px;
            border: 5px solid black;
            margin: 10px;
            box-sizing: border-box;
            background-color: lightblue;
        }
        
        .inline-box {
            display: inline-block;
            width: 100px;
            height: 50px;
            padding: 10px;
            border: 3px solid red;
            margin: 5px;
            box-sizing: content-box;
            background-color: lightgreen;
        }
        
        .hidden-box {
            display: none;
        }
    </style>
</head>
<body>
    <div class="box">This is a block-level box.</div>
    <div class="inline-box">Inline box 1</div>
    <div class="inline-box">Inline box 2</div>
    <div class="hidden-box">This box is hidden.</div>
</body>
</html>
