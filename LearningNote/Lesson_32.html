Advanced Animations

1. Animation Timing Functions

Animation timing functions control the speed curve of the animation. Common timing functions include linear, ease, ease-in, ease-out, and ease-in-out.
  • Syntax:
    animation-timing-function: ease;
  • Example Explanation:
    Different timing functions applied to animations to demonstrate varying speed curves.

2. Animation Delay

Animation delay specifies the amount of time to wait before starting the animation.
  • Syntax:
    animation-delay: 2s;
  • Example Explanation:
    Animation starts after a specified delay.

3. Animation Iteration Count

Animation iteration count defines the number of times the animation should play. It can be a specific number or infinite.
  • Syntax:
    animation-iteration-count: infinite;
  • Example Explanation:
    Animation repeats a specified number of times or infinitely.

4. Animation Direction

Animation direction determines whether the animation should play in reverse on alternate cycles.
  • Syntax:
    animation-direction: alternate;
  • Example Explanation:
    Animation direction is set to play normally, reverse, or alternate.

5. Animation Fill Mode

Animation fill mode specifies how a CSS animation should apply styles to its target before and after it is executing.
  • Syntax:
    animation-fill-mode: forwards;
  • Example Explanation:
    Animation maintains its end state after completion.

Example Code 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Advanced Animations</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            display: flex;
            flex-direction: column;
            align-items: center;
            justify-content: center;
            min-height: 100vh;
            margin: 0;
            background-color: #f0f0f0;
        }

        .box {
            width: 100px;
            height: 100px;
            margin: 20px;
            background-color: #3498db;
            animation-name: move;
            animation-duration: 2s;
            animation-iteration-count: 1;
        }

        @keyframes move {
            from {
                transform: translateX(0);
            }
            to {
                transform: translateX(200px);
            }
        }

        /* Animation Timing Functions example */
        .box-timing {
            animation-timing-function: ease-in-out;
        }

        /* Animation Delay example */
        .box-delay {
            animation-delay: 1s;
        }

        /* Animation Iteration Count example */
        .box-iteration {
            animation-iteration-count: infinite;
        }

        /* Animation Direction example */
        .box-direction {
            animation-direction: alternate;
        }

        /* Animation Fill Mode example */
        .box-fill-mode {
            animation-fill-mode: forwards;
        }
    </style>
</head>
<body>
    <h2>Advanced Animations</h2>

    <div class="box box-timing">Timing Function</div>
    <div class="box box-delay">Delay</div>
    <div class="box box-iteration">Iteration Count</div>
    <div class="box box-direction">Direction</div>
    <div class="box box-fill-mode">Fill Mode</div>
</body>
</html>
