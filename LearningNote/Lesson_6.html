Advanced Flexbox

1. flex-direction

The flex-direction property specifies the direction of the flex container's main axis. This determines the direction in which the flex items are placed in the flex container. 
.flex-direction-row {
    display: flex;
    flex-direction: row; /* Default: left to right */
    border: 1px solid black;
    padding: 10px;
    margin-bottom: 20px;
}

.flex-direction-column {
    display: flex;
    flex-direction: column; /* Top to bottom */
    border: 1px solid black;
    padding: 10px;
    margin-bottom: 20px;
}

.flex-direction-row-reverse {
    display: flex;
    flex-direction: row-reverse; /* Right to left */
    border: 1px solid black;
    padding: 10px;
    margin-bottom: 20px;
}

.flex-direction-column-reverse {
    display: flex;
    flex-direction: column-reverse; /* Bottom to top */
    border: 1px solid black;
    padding: 10px;
    margin-bottom: 20px;
}

2. flex-wrap

The flex-wrap property specifies whether the flex items should wrap or not when the flex container's width or height is insufficient to hold all the items in a single line. 
.flex-wrap {
    display: flex;
    flex-wrap: wrap; /* Items will wrap onto multiple lines */
    border: 1px solid black;
    padding: 10px;
    margin-bottom: 20px;
}

.flex-nowrap {
    display: flex;
    flex-wrap: nowrap; /* Default: items will not wrap */
    border: 1px solid black;
    padding: 10px;
    margin-bottom: 20px;
}

.flex-wrap-reverse {
    display: flex;
    flex-wrap: wrap-reverse; /* Items will wrap onto multiple lines from bottom to top */
    border: 1px solid black;
    padding: 10px;
    margin-bottom: 20px;
}

3. align-content

The align-content property aligns a flex container's lines within it when there is extra space on the cross-axis. This property has no effect on single-line flex containers (i.e., those with flex-wrap: nowrap). 
.align-content-space-between {
    display: flex;
    flex-wrap: wrap;
    align-content: space-between; /* Distributes space between the lines */
    height: 200px; /* Set a height to see the alignment effects */
    border: 1px solid black;
    padding: 10px;
    margin-bottom: 20px;
}

.align-content-center {
    display: flex;
    flex-wrap: wrap;
    align-content: center; /* Packs the lines in the center */
    height: 200px; /* Set a height to see the alignment effects */
    border: 1px solid black;
    padding: 10px;
    margin-bottom: 20px;
}

4. order

The order property specifies the order in which a flex item appears in the flex container. Flex items are displayed in the order they appear in the source code by default. 
.flex-order-container {
    display: flex;
    border: 1px solid black;
    padding: 10px;
    margin-bottom: 20px;
}

.flex-order-item1 {
    order: 2; /* This item will appear second */
    background-color: lightblue;
    padding: 20px;
    margin: 10px;
}

.flex-order-item2 {
    order: 1; /* This item will appear first */
    background-color: lightgreen;
    padding: 20px;
    margin: 10px;
}

.flex-order-item3 {
    order: 3; /* This item will appear third */
    background-color: lightcoral;
    padding: 20px;
    margin: 10px;
}

5. flex-grow, flex-shrink, flex-basis

The flex-grow, flex-shrink, and flex-basis properties are used to control the size and flexibility of flex items.
  • flex-grow: Defines the ability for a flex item to grow if necessary.
  • flex-shrink: Defines the ability for a flex item to shrink if necessary.
  • flex-basis: Defines the default size of an element before the remaining space is distributed. 
.flex-grow-container {
    display: flex;
    border: 1px solid black;
    padding: 10px;
    margin-bottom: 20px;
}

.flex-grow-item {
    flex-grow: 1; /* This item will grow to fill the available space */
    background-color: lightblue;
    padding: 20px;
    margin: 10px;
}

.flex-shrink-container {
    display: flex;
    width: 300px; /* Set a fixed width to see the shrinking effect */
    border: 1px solid black;
    padding: 10px;
    margin-bottom: 20px;
}

.flex-shrink-item1 {
    flex-shrink: 1; /* This item will shrink as necessary */
    background-color: lightgreen;
    padding: 20px;
    margin: 10px;
}

.flex-shrink-item2 {
    flex-shrink: 2; /* This item will shrink twice as much as flex-shrink-item1 */
    background-color: lightcoral;
    padding: 20px;
    margin: 10px;
}

.flex-basis-container {
    display: flex;
    border: 1px solid black;
    padding: 10px;
    margin-bottom: 20px;
}

.flex-basis-item {
    flex-basis: 100px; /* This item will have a base size of 100px */
    background-color: lightyellow;
    padding: 20px;
    margin: 10px;
}
