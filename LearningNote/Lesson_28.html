Custom Fonts and Typography

1. @font-face Rule

The @font-face rule allows you to define custom fonts that can be loaded on a webpage. You can host these fonts on your server or link to them from an external source.
Syntax: 
@font-face {
    font-family: 'CustomFont';
    src: url('customfont.woff2') format('woff2'),
         url('customfont.woff') format('woff');
    font-weight: normal;
    font-style: normal;
}

2. Google Fonts

Google Fonts provides a large library of free fonts that can be easily integrated into your website.
Usage:
  1. Go to Google Fonts.
  2. Choose the fonts you want to use.
  3. Embed the font in your HTML by copying the provided <link> tag.
Example: 
<link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap">

3. Web Safe Fonts

Web Safe Fonts are a set of fonts that are commonly available on most devices. Examples include Arial, Verdana, Times New Roman, and Courier New. Using web safe fonts ensures consistency across different platforms.
Example: 
body {
    font-family: 'Arial', 'Helvetica', sans-serif;
}

4. Font Weight and Style

Font weight specifies the thickness of the font. Common values include normal, bold, bolder, and numerical values from 100 to 900. Font style specifies whether the font is normal, italic, or oblique.
Example: 
h1 {
    font-weight: bold;
    font-style: italic;
}

5. Text Shadow

Text shadow adds a shadow effect to text. It can enhance the readability and aesthetics of your text.
Syntax: 
text-shadow: horizontal-offset vertical-offset blur-radius color;
Example: 
h1 {
    text-shadow: 2px 2px 5px rgba(0,0,0,0.5);
}

Example Code 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Custom Fonts and Typography</title>
    
    <!-- Google Fonts -->
    <link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap">
    
    <!-- Custom Font using @font-face -->
    <style>
        @font-face {
            font-family: 'CustomFont';
            src: url('https://example.com/fonts/customfont.woff2') format('woff2'),
                 url('https://example.com/fonts/customfont.woff') format('woff');
            font-weight: normal;
            font-style: normal;
        }
        
        body {
            font-family: 'Roboto', 'Arial', sans-serif;
            margin: 0;
            padding: 0;
            background-color: #f4f4f4;
            color: #333;
        }
        
        h1 {
            font-family: 'CustomFont', 'Roboto', sans-serif;
            font-weight: 700;
            font-style: italic;
            text-shadow: 2px 2px 5px rgba(0, 0, 0, 0.5);
            text-align: center;
            margin-top: 50px;
        }
        
        p {
            font-weight: 400;
            font-size: 1.2em;
            text-align: center;
            margin: 20px;
        }
    </style>
</head>
<body>
    <h1>Custom Fonts and Typography</h1>
    <p>This is a paragraph using Google Fonts (Roboto) with normal weight.</p>
    <p style="font-family: 'Arial', sans-serif;">This paragraph uses a web safe font (Arial).</p>
    <p style="font-weight: 700;">This paragraph is bold.</p>
    <p style="font-style: italic;">This paragraph is italic.</p>
    <p style="text-shadow: 2px 2px 5px rgba(0,0,0,0.3);">This paragraph has a text shadow.</p>
</body>
</html>
