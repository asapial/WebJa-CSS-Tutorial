Icon Fonts and SVG

1. Font Awesome Icons

Font Awesome is a popular icon set and toolkit.
Using Font Awesome Icons:
  1. Include the Font Awesome CDN in your HTML: 
    <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0-beta3/css/all.min.css">
  2. Use the icons in your HTML:
    <i class="fas fa-camera"></i>
<i class="fas fa-heart"></i>

2. IcoMoon

IcoMoon is another popular tool for custom icon sets and icon fonts.
Using IcoMoon:
  1. Generate and download your icon font from IcoMoon.
  2. Include the IcoMoon CSS in your HTML:
    <link rel="stylesheet" href="path/to/icomoon/style.css">
  3. Use the icons in your HTML:
    <i class="icon-camera"></i>
<i class="icon-heart"></i>

3. Inline SVG

SVG (Scalable Vector Graphics) can be used directly in HTML for sharp, scalable images.
Using Inline SVG: 
<svg width="100" height="100" viewBox="0 0 100 100">
  <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
</svg>

4. SVG Backgrounds

SVG images can be used as backgrounds in CSS.
Using SVG Backgrounds: 
.svg-background {
    width: 100px;
    height: 100px;
    background: url('path/to/image.svg');
}

5. Styling SVG with CSS

SVG elements can be styled with CSS, similar to HTML elements.
Styling Inline SVG: 
<svg width="100" height="100" viewBox="0 0 100 100" class="styled-svg">
  <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
</svg>

<style>
  .styled-svg circle {
      fill: blue;
      stroke: green;
      stroke-width: 5;
  }
</style>

Example Code 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Icon Fonts and SVG Example</title>
    <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0-beta3/css/all.min.css">
    <link rel="stylesheet" href="path/to/icomoon/style.css">
    <style>
        .svg-background {
            width: 100px;
            height: 100px;
            background: url('path/to/image.svg');
        }

        .styled-svg circle {
            fill: blue;
            stroke: green;
            stroke-width: 5;
        }
    </style>
</head>
<body>
    <h2>Font Awesome Icons</h2>
    <i class="fas fa-camera"></i>
    <i class="fas fa-heart"></i>

    <h2>IcoMoon Icons</h2>
    <i class="icon-camera"></i>
    <i class="icon-heart"></i>

    <h2>Inline SVG</h2>
    <svg width="100" height="100" viewBox="0 0 100 100">
        <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
    </svg>

    <h2>SVG Background</h2>
    <div class="svg-background"></div>

    <h2>Styling SVG with CSS</h2>
    <svg width="100" height="100" viewBox="0 0 100 100" class="styled-svg">
        <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
    </svg>
</body>
</html>
