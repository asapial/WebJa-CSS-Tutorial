Content and Media Queries

1. min-width and max-width

The min-width and max-width media queries apply styles based on the minimum or maximum width of the viewport.
  • Syntax:
    @media (min-width: 600px) { ... }
@media (max-width: 600px) { ... }
  • Example Explanation:
    Change the background color based on the viewport width.

2. min-height and max-height

The min-height and max-height media queries apply styles based on the minimum or maximum height of the viewport.
  • Syntax:
    @media (min-height: 800px) { ... }
@media (max-height: 800px) { ... }
  • Example Explanation:
    Change the font size based on the viewport height.

3. Orientation Media Queries

The orientation media query applies styles based on the orientation of the viewport (portrait or landscape).
  • Syntax:
    @media (orientation: portrait) { ... }
@media (orientation: landscape) { ... }
  • Example Explanation:
    Apply different padding based on the orientation.

4. Aspect Ratio Media Queries

The aspect-ratio media query applies styles based on the aspect ratio of the viewport.
  • Syntax:
    @media (aspect-ratio: 16/9) { ... }
@media (min-aspect-ratio: 4/3) { ... }
  • Example Explanation:
    Change the border color based on the aspect ratio.

5. Dark Mode Media Queries

The prefers-color-scheme media query applies styles based on the user's preferred color scheme (light or dark mode).
  • Syntax:
    @media (prefers-color-scheme: dark) { ... }
@media (prefers-color-scheme: light) { ... }
  • Example Explanation:
    Change the text color based on the user's color scheme preference.

Example Code 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Content and Media Queries</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            margin: 0;
            padding: 0;
            display: flex;
            flex-direction: column;
            align-items: center;
            justify-content: center;
            min-height: 100vh;
        }

        .box {
            width: 80%;
            max-width: 800px;
            padding: 20px;
            margin: 20px;
            background-color: #f0f0f0;
            border: 2px solid #ccc;
            text-align: center;
        }

        /* min-width and max-width example */
        @media (min-width: 600px) {
            .box {
                background-color: #d1f7c4; /* Light green */
            }
        }

        @media (max-width: 599px) {
            .box {
                background-color: #f7d1d1; /* Light red */
            }
        }

        /* min-height and max-height example */
        @media (min-height: 800px) {
            .box {
                font-size: 1.5em;
            }
        }

        @media (max-height: 799px) {
            .box {
                font-size: 1em;
            }
        }

        /* Orientation Media Queries example */
        @media (orientation: portrait) {
            .box {
                padding: 40px;
            }
        }

        @media (orientation: landscape) {
            .box {
                padding: 20px;
            }
        }

        /* Aspect Ratio Media Queries example */
        @media (aspect-ratio: 16/9) {
            .box {
                border-color: #3498db; /* Blue */
            }
        }

        @media (min-aspect-ratio: 4/3) {
            .box {
                border-color: #2ecc71; /* Green */
            }
        }

        /* Dark Mode Media Queries example */
        @media (prefers-color-scheme: dark) {
            body {
                background-color: #333;
                color: #fff;
            }

            .box {
                background-color: #444;
                border-color: #666;
            }
        }

        @media (prefers-color-scheme: light) {
            body {
                background-color: #fff;
                color: #000;
            }

            .box {
                background-color: #f0f0f0;
                border-color: #ccc;
            }
        }
    </style>
</head>
<body>
    <div class="box">
        <h2>Responsive Design with Media Queries</h2>
        <p>This box changes style based on various media queries.</p>
    </div>
</body>
</html>
