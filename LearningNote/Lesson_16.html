CSS Variables

1. Defining Variables

CSS variables are defined using the -- syntax within a selector, typically within the :root selector to make them globally available.
  • Syntax: 
    :root {
    --primary-color: #3498db;
    --secondary-color: #2ecc71;
    --font-size: 16px;
}

2. Using Variables

To use a CSS variable, you use the var() function.
  • Syntax: 
    body {
    background-color: var(--primary-color);
    color: var(--secondary-color);
    font-size: var(--font-size);
}

3. Variables in Media Queries

CSS variables can be used and redefined within media queries to create responsive designs.
  • Example: 
    :root {
    --font-size: 16px;
}

body {
    font-size: var(--font-size);
}

@media (max-width: 600px) {
    :root {
        --font-size: 14px;
    }
}

4. Variables with JavaScript

CSS variables can be manipulated using JavaScript, which allows dynamic changes to styles.
  • Example: 
    <div class="box">Click me to change color</div>
<button onclick="changeColor()">Change Color</button>

<script>
    function changeColor() {
        document.documentElement.style.setProperty('--primary-color', '#e74c3c');
    }
</script>

    :root {
    --primary-color: #3498db;
}

.box {
    width: 100px;
    height: 100px;
    background-color: var(--primary-color);
}

5. Fallback Values

Fallback values can be provided for CSS variables to ensure a value is used if the variable is not defined.
  • Syntax: 
    .element {
    color: var(--undefined-color, black);
}
