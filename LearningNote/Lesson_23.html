CSS Units

1. Absolute Units

Absolute units are fixed and do not change with respect to other elements.
  • px: Pixels
  • pt: Points (1pt = 1/72 of an inch)
  • cm: Centimeters
  • mm: Millimeters
  • in: Inches (1in = 96px = 2.54cm)
.absolute-units {
    width: 100px;
    height: 50px;
    border: 1px solid black;
    margin-bottom: 10px;
}

2. Relative Units

Relative units are flexible and change relative to other elements or the viewport.
  • em: Relative to the font-size of the element (2em means 2 times the size of the current font)
  • rem: Relative to the font-size of the root element (html element)
  • %: Relative to the parent element
  • vw: Relative to 1% of the viewport’s width
  • vh: Relative to 1% of the viewport’s height
.relative-units {
    width: 50%;
    height: 10em;
    font-size: 2rem;
    border: 1px solid black;
    margin-bottom: 10px;
}

3. Differences Between px, em, and rem

  • px: Fixed size, absolute unit.
  • em: Relative to the parent element's font-size.
  • rem: Relative to the root element's font-size.
.differences {
    font-size: 16px;
    padding: 10px;
}

.differences .px-example {
    font-size: 20px; /* Always 20px */
}

.differences .em-example {
    font-size: 2em; /* 2 times the size of the parent font-size (32px) */
}

.differences .rem-example {
    font-size: 2rem; /* 2 times the root font-size (32px if root is 16px) */
}

4. Using Viewport Units

Viewport units (vw and vh) are based on the size of the viewport.
  • vw: 1% of the viewport's width.
  • vh: 1% of the viewport's height.
.viewport-units {
    width: 50vw; /* 50% of the viewport's width */
    height: 50vh; /* 50% of the viewport's height */
    border: 1px solid black;
}

5. Combining Units

Combining different units can provide more responsive and adaptive designs. 
.combining-units {
    font-size: 2rem; /* Relative to the root element */
    padding: 10px 5%; /* 10px top and bottom, 5% left and right */
    width: calc(100% - 20px); /* Width minus fixed padding */
    height: 10em; /* Relative to the element’s font-size */
    border: 1px solid black;
}

<!DOCTYPE html>
<html lang="en">
<head>

            display: flex;
            flex-direction: column;

        /* Example 1: Absolute Units */
        .absolute-units {
            width: 100px;
            height: 50px;
            border: 1px solid black;
            margin-bottom: 10px;
        }

        /* Example 2: Relative Units */
        .relative-units {
            width: 50%;
            height: 10em;
            font-size: 2rem;
            border: 1px solid black;
            margin-bottom: 10px;
        }

        /* Example 3: Differences Between px, em, and rem */
        .differences {
            font-size: 16px;
            padding: 10px;
        }

        .differences .px-example {
            font-size: 20px; /* Always 20px */
        }

        .differences .em-example {
            font-size: 2em; /* 2 times the size of the parent font-size (32px) */
        }

        .differences .rem-example {
            font-size: 2rem; /* 2 times the root font-size (32px if root is 16px) */
        }

        /* Example 4: Using Viewport Units */
        .viewport-units {
            width: 50vw; /* 50% of the viewport's width */
            height: 50vh; /* 50% of the viewport's height */
            border: 1px solid black;
        }

        /* Example 5: Combining Units */
        .combining-units {
            font-size: 2rem; /* Relative to the root element */
            padding: 10px 5%; /* 10px top and bottom, 5% left and right */
            width: calc(100% - 20px); /* Width minus fixed padding */
            height: 10em; /* Relative to the element’s font-size */
            border: 1px solid black;
        }
    </style>
</head>
