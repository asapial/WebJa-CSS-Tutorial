Grid Layout Basics

1. Introduction to CSS Grid

CSS Grid Layout is a two-dimensional layout system for the web. It allows you to design web pages using a grid system with rows and columns, making it easier to create complex and responsive web designs.

2. Grid Container

A grid container is an element on which the display property is set to grid or inline-grid. This container establishes a grid formatting context for its direct children. 
.grid-container {
    display: grid;
    border: 1px solid black;
    padding: 10px;
    margin-bottom: 20px;
}

3. Grid Items

Grid items are the direct children of a grid container. They can be manipulated using various grid properties. 
.grid-item {
    background-color: lightblue;
    padding: 20px;
    margin: 10px;
    border: 1px solid darkblue;
}

4. grid-template-columns, grid-template-rows

The grid-template-columns and grid-template-rows properties define the columns and rows of the grid. You can specify the size of the columns and rows using different units like pixels, percentages, fr units, etc. 
.grid-template-columns-rows {
    display: grid;
    grid-template-columns: 1fr 2fr 1fr; /* Three columns with different fractions */
    grid-template-rows: 100px 200px; /* Two rows with fixed heights */
    border: 1px solid black;
    padding: 10px;
    margin-bottom: 20px;
}

5. grid-gap

The grid-gap property specifies the gap (space) between rows and columns. 
.grid-gap {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr; /* Three equal columns */
    grid-template-rows: 100px 100px; /* Two equal rows */
    grid-gap: 10px; /* Gap between rows and columns */
    border: 1px solid black;
    padding: 10px;
    margin-bottom: 20px;
}

Example Code

Here is an example demonstrating the basics of CSS Grid: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Grid Layout Basics</title>
    <style>
        /* Grid container */
        .grid-container {
            display: grid;
            border: 1px solid black;
            padding: 10px;
            margin-bottom: 20px;
        }

        /* Grid items */
        .grid-item {
            background-color: lightblue;
            padding: 20px;
            margin: 10px;
            border: 1px solid darkblue;
        }

        /* Grid template columns and rows */
        .grid-template-columns-rows {
            display: grid;
            grid-template-columns: 1fr 2fr 1fr; /* Three columns with different fractions */
            grid-template-rows: 100px 200px; /* Two rows with fixed heights */
            border: 1px solid black;
            padding: 10px;
            margin-bottom: 20px;
        }

        /* Grid gap */
        .grid-gap {
            display: grid;
            grid-template-columns: 1fr 1fr 1fr; /* Three equal columns */
            grid-template-rows: 100px 100px; /* Two equal rows */
            grid-gap: 10px; /* Gap between rows and columns */
            border: 1px solid black;
            padding: 10px;
            margin-bottom: 20px;
        }
    </style>
</head>
<body>
    <!-- Grid container example -->
    <div class="grid-container">
        <div class="grid-item">Item 1</div>
        <div class="grid-item">Item 2</div>
        <div class="grid-item">Item 3</div>
        <div class="grid-item">Item 4</div>
        <div class="grid-item">Item 5</div>
        <div class="grid-item">Item 6</div>
    </div>

    <!-- Grid template columns and rows example -->
    <div class="grid-template-columns-rows">
        <div class="grid-item">Item 1</div>
        <div class="grid-item">Item 2</div>
        <div class="grid-item">Item 3</div>
        <div class="grid-item">Item 4</div>
        <div class="grid-item">Item 5</div>
        <div class="grid-item">Item 6</div>
    </div>

    <!-- Grid gap example -->
    <div class="grid-gap">
        <div class="grid-item">Item 1</div>
        <div class="grid-item">Item 2</div>
        <div class="grid-item">Item 3</div>
        <div class="grid-item">Item 4</div>
        <div class="grid-item">Item 5</div>
        <div class="grid-item">Item 6</div>
    </div>
</body>
</html>
