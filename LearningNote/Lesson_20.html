Advanced Selectors

CSS selectors are patterns used to select the elements you want to style. Advanced selectors allow for more complex and specific selections beyond simple class or ID selectors.

1. Attribute Selectors

Attribute selectors allow you to select elements based on the presence or value of an attribute.
Syntax: 
/* Select elements with a specific attribute */
[attribute]

/* Select elements with a specific attribute and value */
[attribute="value"]

/* Select elements with an attribute value starting with a specific value */
[attribute^="value"]

/* Select elements with an attribute value ending with a specific value */
[attribute$="value"]

/* Select elements with an attribute value containing a specific value */
[attribute*="value"]

2. Child Combinators

Child combinators select elements that are direct children of a specified element.
Syntax: 
parent > child {
    /* styles */
}

3. Sibling Combinators

Sibling combinators select elements based on their sibling relationship.
Adjacent Sibling Selector: 
/* Selects elements that are immediately preceded by a specified element */
previous + next {
    /* styles */
}
General Sibling Selector: 
/* Selects elements that are preceded by a specified element, but not necessarily immediately */
previous ~ next {
    /* styles */
}

4. Descendant Selectors

Descendant selectors select elements that are descendants of a specified element, regardless of their immediate parent.
Syntax: 
ancestor descendant {
    /* styles */
}

5. Universal Selector

The universal selector selects all elements on the page.
Syntax: 
* {
    /* styles */
}

Example Code 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Advanced CSS Selectors</title>
    <style>
        /* Attribute Selectors */
        [data-type] {
            border: 2px solid blue;
            padding: 10px;
            margin: 10px;
        }

        [data-type="primary"] {
            background-color: lightblue;
        }

        [data-type^="sec"] {
            font-weight: bold;
        }

        [data-type$="dary"] {
            color: green;
        }

        [data-type*="mid"] {
            text-decoration: underline;
        }

        /* Child Combinators */
        .parent > .child {
            background-color: lightgreen;
            padding: 10px;
            margin: 10px;
        }

        /* Sibling Combinators */
        .item + .item {
            background-color: lightcoral;
            padding: 10px;
            margin: 10px;
        }

        .group ~ .group {
            border: 2px solid orange;
            padding: 10px;
            margin: 10px;
        }

        /* Descendant Selectors */
        .ancestor .descendant {
            color: blue;
        }

        /* Universal Selector */
        * {
            margin: 0;
            padding: 0;
            box-sizing: border-box;
        }

        body {
            font-family: Arial, sans-serif;
            padding: 20px;
        }
    </style>
</head>
<body>
    <h1>Advanced CSS Selectors Example</h1>
    <section>
        <h2>Attribute Selectors</h2>
        <p data-type="primary">Primary Type</p>
        <p data-type="secondary">Secondary Type</p>
        <p data-type="tertiary">Tertiary Type</p>
        <p data-type="midway">Midway Type</p>
    </section>

    <section>
        <h2>Child Combinators</h2>
        <div class="parent">
            <div class="child">Direct Child</div>
            <div>Not a Direct Child
                <div class="child">Not Direct</div>
            </div>
        </div>
    </section>

    <section>
        <h2>Sibling Combinators</h2>
        <div class="item">First Item</div>
        <div class="item">Next Item</div>
        <div class="item">Another Item</div>
        <div class="group">First Group</div>
        <div class="other">Other Element</div>
        <div class="group">Next Group</div>
    </section>

    <section>
        <h2>Descendant Selectors</h2>
        <div class="ancestor">
            <p class="descendant">Descendant</p>
            <div>
                <p class="descendant">Nested Descendant</p>
            </div>
        </div>
    </section>

    <section>
        <h2>Universal Selector</h2>
        <p>This paragraph and the entire page have no default margin or padding.</p>
    </section>
</body>
</html>
