CSS Shapes

1. Creating Circles

You can create circles in CSS using border-radius property. Setting it to 50% will make a square element into a circle.
Example:
.circle {
    width: 100px;
    height: 100px;
    background-color: #3498db;
    border-radius: 50%; /* Makes the element a circle */
}

2. Creating Triangles

You can create triangles in CSS using borders of zero width and adjusting border colors for different sides.
Example:
.triangle {
    width: 0;
    height: 0;
    border-left: 50px solid transparent;
    border-right: 50px solid transparent;
    border-bottom: 100px solid #e74c3c; /* Adjust color for triangle */
}

3. Creating Polygons

Polygons are created using clip-path property. You can define the shape of the polygon using polygon() function.
Example:
.polygon {
    width: 200px;
    height: 200px;
    background-color: #2ecc71;
    clip-path: polygon(50% 0%, 100% 50%, 50% 100%, 0% 50%);
}

4. CSS Shapes with Clip-path

Clip-path allows creating complex shapes by defining a clipping region. It can be used with various shapes defined by polygon(), circle(), ellipse(), etc.
Example:
.custom-shape {
    width: 200px;
    height: 200px;
    background-color: #f39c12;
    clip-path: polygon(25% 0%, 100% 0%, 75% 100%, 0% 100%);
}

5. CSS Shapes with Border-radius

Border-radius can create rounded shapes and also complex shapes when combined with pseudo-elements and positioning.
Example:
.complex-shape {
    width: 150px;
    height: 150px;
    background-color: #9b59b6;
    border-radius: 50% 0 50% 50%; /* Creates a rounded shape */
}
