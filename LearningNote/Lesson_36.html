CSS Counters

1. Using CSS Counters

CSS Counters allow you to create custom counters (like list item numbers) in your CSS without relying on JavaScript. They are particularly useful for ordered lists, nested lists, and other elements that need incremental numbering.

Key Points:

  • Counters are created and manipulated using counter-reset, counter-increment, and content properties.
  • counter-reset initializes or resets a counter.
  • counter-increment increments a counter.
  • content is used with the ::before or ::after pseudo-elements to display the counter value.

Example: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Using CSS Counters</title>
    <style>
        .counter-container {
            counter-reset: my-counter; /* Initialize the counter */
        }

        .counter-item {
            counter-increment: my-counter; /* Increment the counter */
        }

        .counter-item::before {
            content: counter(my-counter) ". "; /* Display the counter */
            font-weight: bold;
        }
    </style>
</head>
<body>
    <div class="counter-container">
        <div class="counter-item">First item</div>
        <div class="counter-item">Second item</div>
        <div class="counter-item">Third item</div>
    </div>
</body>
</html>

2. Counter Increment

counter-increment increments the specified counter value. This is typically used within the elements that you want to number.

Example: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Counter Increment</title>
    <style>
        .counter-container {
            counter-reset: my-counter;
        }

        .counter-item {
            counter-increment: my-counter 2; /* Increment the counter by 2 */
        }

        .counter-item::before {
            content: counter(my-counter) ". ";
            font-weight: bold;
        }
    </style>
</head>
<body>
    <div class="counter-container">
        <div class="counter-item">First item</div>
        <div class="counter-item">Second item</div>
        <div class="counter-item">Third item</div>
    </div>
</body>
</html>

3. Counter Reset

counter-reset initializes or resets a counter. You can use this property to start a counter at a specific value or to reset it at a certain point in the document.

Example: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Counter Reset</title>
    <style>
        .section {
            counter-reset: section-counter;
        }

        .section-title {
            counter-increment: section-counter;
        }

        .section-title::before {
            content: "Section " counter(section-counter) ": ";
            font-weight: bold;
        }
    </style>
</head>
<body>
    <div class="section">
        <h2 class="section-title">Introduction</h2>
        <p>This is the introduction section.</p>
    </div>
    <div class="section">
        <h2 class="section-title">Main Content</h2>
        <p>This is the main content section.</p>
    </div>
    <div class="section">
        <h2 class="section-title">Conclusion</h2>
        <p>This is the conclusion section.</p>
    </div>
</body>
</html>

4. Nested Counters

Nested counters are useful for complex documents like legal documents or scientific papers that require hierarchical numbering. You can reset and increment multiple counters to achieve this.

Example: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Nested Counters</title>
    <style>
        .chapter {
            counter-reset: section;
        }

        .chapter-title {
            counter-increment: chapter;
        }

        .chapter-title::before {
            content: "Chapter " counter(chapter) ": ";
            font-weight: bold;
        }

        .section {
            counter-reset: subsection;
        }

        .section-title {
            counter-increment: section;
        }

        .section-title::before {
            content: counter(chapter) "." counter(section) " ";
            font-weight: bold;
        }

        .subsection-title {
            counter-increment: subsection;
        }

        .subsection-title::before {
            content: counter(chapter) "." counter(section) "." counter(subsection) " ";
            font-weight: bold;
        }
    </style>
</head>
<body>
    <div class="chapter">
        <h1 class="chapter-title">Introduction</h1>
        <div class="section">
            <h2 class="section-title">Background</h2>
            <p>This is the background section.</p>
            <h3 class="subsection-title">History</h3>
            <p>This is the history subsection.</p>
        </div>
        <div class="section">
            <h2 class="section-title">Objectives</h2>
            <p>This is the objectives section.</p>
            <h3 class="subsection-title">Goals</h3>
            <p>This is the goals subsection.</p>
        </div>
    </div>
</body>
</html>

5. Styling Counters

Counters can be styled using various CSS properties to make them visually appealing and to fit the design of your document.

Example: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Styling Counters</title>
    <style>
        .list {
            counter-reset: list-counter;
            list-style: none;
        }

        .list-item {
            counter-increment: list-counter;
            margin: 10px 0;
        }

        .list-item::before {
            content: counter(list-counter) ". ";
            font-weight: bold;
            color: #3498db;
            font-size: 1.2em;
            margin-right: 5px;
        }
    </style>
</head>
<body>
    <ul class="list">
        <li class="list-item">First item</li>
        <li class="list-item">Second item</li>
        <li class="list-item">Third item</li>
    </ul>
</body>
</html>
