CSS Grid Advanced

1. Grid Auto Placement

Grid auto placement allows the grid items to be placed automatically within the grid container without explicitly specifying their positions.
Example Code: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Grid Auto Placement</title>
    <style>
        .grid-container {
            display: grid;
            grid-template-columns: repeat(3, 1fr);
            grid-gap: 10px;
        }

        .grid-item {
            background-color: #4CAF50;
            color: white;
            padding: 20px;
            text-align: center;
        }
    </style>
</head>
<body>
    <h2>Grid Auto Placement</h2>
    <div class="grid-container">
        <div class="grid-item">1</div>
        <div class="grid-item">2</div>
        <div class="grid-item">3</div>
        <div class="grid-item">4</div>
        <div class="grid-item">5</div>
        <div class="grid-item">6</div>
    </div>
</body>
</html>

2. minmax()

The minmax() function defines a size range with a minimum and maximum value.
Example Code: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>minmax()</title>
    <style>
        .grid-container {
            display: grid;
            grid-template-columns: repeat(3, minmax(100px, 1fr));
            grid-gap: 10px;
        }

        .grid-item {
            background-color: #4CAF50;
            color: white;
            padding: 20px;
            text-align: center;
        }
    </style>
</head>
<body>
    <h2>minmax()</h2>
    <div class="grid-container">
        <div class="grid-item">1</div>
        <div class="grid-item">2</div>
        <div class="grid-item">3</div>
        <div class="grid-item">4</div>
        <div class="grid-item">5</div>
        <div class="grid-item">6</div>
    </div>
</body>
</html>

auto-fill and auto-fit automatically adjust the number of columns based on the available space.
Example Code: 
<!DOCTYPE html>

            background-color: #4CAF50;

            padding: 20px;
            text-align: center;

        <div class="grid-item">3</div>

        <div class="grid-item">5</div>
        <div class="grid-item">6</div>
    </div>
</body>
</html>

4. Fractional Units (fr)

The fr unit allocates a fraction of the available space in the grid container.
Example Code: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Fractional Units (fr)</title>
    <style>
        .grid-container {
            display: grid;
            grid-template-columns: 1fr 2fr 1fr;
            grid-gap: 10px;
        }

        .grid-item {
            background-color: #4CAF50;
            color: white;
            padding: 20px;
            text-align: center;
        }
    </style>
</head>
<body>
    <h2>Fractional Units (fr)</h2>
    <div class="grid-container">
        <div class="grid-item">1</div>
        <div class="grid-item">2</div>
        <div class="grid-item">3</div>
    </div>
</body>
</html>

5. Nested Grids

Grid items can themselves be grid containers, creating nested grids.
Example Code: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Nested Grids</title>
    <style>
        .outer-grid {
            display: grid;
            grid-template-columns: 1fr 1fr;
            grid-gap: 10px;
        }

        .inner-grid {
            display: grid;
            grid-template-columns: repeat(2, 1fr);
            grid-gap: 5px;
        }

        .grid-item {
            background-color: #4CAF50;
            color: white;
            padding: 20px;
            text-align: center;
        }
    </style>
</head>
<body>
    <h2>Nested Grids</h2>
    <div class="outer-grid">
        <div class="grid-item">1</div>
        <div class="inner-grid">
            <div class="grid-item">2.1</div>
            <div class="grid-item">2.2</div>
        </div>
    </div>
</body>
</html>

Explanation

  1. Grid Auto Placement: Items are placed in the grid without specifying their exact position, filling the grid based on the defined columns.
  2. minmax(): Sets a range for a grid track size, ensuring a minimum and maximum size.
  3. auto-fill and auto-fit: Adjust the number of columns based on the container's width and the size of the grid items.
  4. Fractional Units (fr): Allocate available space proportionally based on the number of fr units specified.
  5. Nested Grids: Allows for complex layouts by nesting grid containers within grid items.
These advanced grid techniques enable powerful and flexible layouts in modern web design.

Example Code 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Advanced CSS Grid Example</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            background-color: #f4f4f4;
            margin: 0;
            padding: 20px;
        }

        .grid-container {
            display: grid;
            grid-template-columns: repeat(auto-fill, minmax(150px, 1fr));
            gap: 10px;
            background-color: #fff;
            padding: 10px;
            border-radius: 8px;
            box-shadow: 0 2px 5px rgba(0,0,0,0.1);
        }

        .grid-item {
            background-color: #4CAF50;
            color: white;
            padding: 20px;
            font-size: 1.2em;
            text-align: center;
            border-radius: 4px;
        }

        .nested-grid {
            display: grid;
            grid-template-columns: 1fr 1fr;
            gap: 5px;
        }

        .nested-item {
            background-color: #ff9800;
            padding: 10px;
            border-radius: 4px;
        }
    </style>
</head>
<body>
    <h1>Advanced CSS Grid Example</h1>
    <p>This example demonstrates advanced CSS Grid features such as auto-placement, minmax(), auto-fill, auto-fit, fractional units (fr), and nested grids.</p>

    <div class="grid-container">
        <!-- Grid Auto Placement and minmax() -->
        <div class="grid-item">Item 1</div>
        <div class="grid-item">Item 2</div>
        <div class="grid-item">Item 3</div>
        <div class="grid-item">Item 4</div>
        <div class="grid-item">Item 5</div>
        <div class="grid-item">Item 6</div>

        <!-- Nested Grid -->
        <div class="grid-item">
            Nested Grid
            <div class="nested-grid">
                <div class="nested-item">Nested 1</div>
                <div class="nested-item">Nested 2</div>
                <div class="nested-item">Nested 3</div>
                <div class="nested-item">Nested 4</div>
            </div>
        </div>

        <!-- Fractional Units (fr) -->
        <div class="grid-item" style="grid-column: span 2;">
            <div style="display: grid; grid-template-columns: 2fr 1fr; gap: 10px;">
                <div class="nested-item">2fr</div>
                <div class="nested-item">1fr</div>
            </div>
        </div>
    </div>
</body>
</html>
