Responsive Design Basics

1. Media Queries

Media queries allow you to apply CSS styles based on the characteristics of the device, such as its width, height, and orientation.
  • Syntax:
    @media (max-width: 600px) {
    .container {
        background-color: lightblue;
    }
}
    This example applies styles to elements with the class container when the viewport width is 600 pixels or less.
  • Example:
    <div class="container">Resize the window to see the effect!</div>

    .container {
    width: 100%;
    padding: 20px;
    background-color: lightcoral;
}

@media (max-width: 600px) {
    .container {
        background-color: lightblue;
    }
}

2. Viewport Units (vw, vh)

Viewport units are relative to the size of the viewport (the browser window). vw stands for viewport width, and vh stands for viewport height.
  • Usage:
    • 1vw is 1% of the viewport width.
    • 1vh is 1% of the viewport height.
  • Example:
    <div class="box">Viewport Units</div>

    .box {
    width: 50vw; /* 50% of the viewport width */
    height: 50vh; /* 50% of the viewport height */
    background-color: lightgreen;
}

3. Responsive Typography

Responsive typography adjusts the font size based on the screen size.
  • Example: 
    <div class="text">Responsive Text</div>

    .text {
    font-size: 2vw; /* 2% of the viewport width */
}

@media (max-width: 600px) {
    .text {
        font-size: 4vw; /* 4% of the viewport width */
    }
}

4. Fluid Layouts

Fluid layouts use percentages instead of fixed units like pixels, allowing elements to resize based on the parent container's size.
  • Example: 
    <div class="fluid-container">
    <div class="fluid-box">Fluid Box 1</div>
    <div class="fluid-box">Fluid Box 2</div>
</div>

    .fluid-container {
    width: 100%;
    display: flex;
}

.fluid-box {
    width: 50%; /* Each box takes 50% of the container's width */
    padding: 20px;
    background-color: lightcoral;
    box-sizing: border-box;
}

5. Mobile-first Design

Mobile-first design is an approach where you design for smaller screens first and then use media queries to add styles for larger screens.
  • Example: 
    <div class="mobile-first">Mobile-first Design</div>

    .mobile-first {
    background-color: lightblue;
    padding: 20px;
}

@media (min-width: 768px) {
    .mobile-first {
        background-color: lightcoral;
    }
}
