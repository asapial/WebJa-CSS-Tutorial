Text and Font Styling

1. Text Color

The color property is used to set the color of the text. You can use color names, hex values, RGB, RGBA, HSL, or HSLA values. 
p {
  color: red; /* Color name */
}

h1 {
  color: #ff0000; /* Hex value */
}

span {
  color: rgb(255, 0, 0); /* RGB value */
}

a {
  color: rgba(255, 0, 0, 0.5); /* RGBA value (with alpha transparency) */
}

div {
  color: hsl(0, 100%, 50%); /* HSL value */
}

2. Text Alignment

The text-align property specifies the horizontal alignment of text within an element. Common values are left, right, center, and justify. 
h1 {
  text-align: center; /* Centers the text */
}

p {
  text-align: justify; /* Stretches the lines so that each line has equal width */
}

3. Text Decoration

The text-decoration property is used to add or remove decorations from text, such as underlines, overlines, or line-throughs. 
a {
  text-decoration: none; /* Removes the underline from links */
}

u {
  text-decoration: underline; /* Underlines the text */
}

del {
  text-decoration: line-through; /* Strikes through the text */
}

p.overline {
  text-decoration: overline; /* Adds a line above the text */
}

4. Font Family

The font-family property specifies the font for an element. You can list multiple fonts as a fallback system. 
body {
  font-family: Arial, sans-serif; /* Uses Arial, or a default sans-serif font if Arial is unavailable */
}

h1 {
  font-family: "Times New Roman", Times, serif; /* Uses Times New Roman, Times, or a default serif font */
}

5. Font Size

The font-size property sets the size of the text. You can use various units such as pixels (px), ems (em), rems (rem), percentages (%), and more. 
p {
  font-size: 16px; /* Sets the font size to 16 pixels */
}

h1 {
  font-size: 2em; /* Sets the font size to 2 times the element's current font size */
}

body {
  font-size: 100%; /* Sets the font size to 100% of the default browser font size */
}

span {
  font-size: 1.5rem; /* Sets the font size to 1.5 times the root element's font size */
}
