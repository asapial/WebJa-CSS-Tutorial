Theming and Customization

Theming and customization are crucial for creating user-friendly and personalized web experiences. This lesson covers various aspects of theming, including light and dark themes, user preference media queries, CSS variables for theming, theme switchers, and theming with preprocessors.

1. Light and Dark Themes

Description:
Light and dark themes allow users to switch between a light background with dark text and a dark background with light text. This improves readability and user comfort in different lighting conditions.
Example: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Light and Dark Themes Example</title>
    <style>
        body.light-theme {
            background-color: #ffffff;
            color: #000000;
        }

        body.dark-theme {
            background-color: #333333;
            color: #ffffff;
        }

        .theme-toggle {
            margin: 20px;
            padding: 10px;
            cursor: pointer;
            background-color: #007BFF;
            color: white;
            border: none;
            border-radius: 5px;
        }
    </style>
</head>
<body class="light-theme">
    <h2>Light and Dark Themes Example</h2>
    <button class="theme-toggle">Toggle Theme</button>

    <script>
        const toggleButton = document.querySelector('.theme-toggle');
        toggleButton.addEventListener('click', () => {
            document.body.classList.toggle('dark-theme');
            document.body.classList.toggle('light-theme');
        });
    </script>
</body>
</html>

2. User Preference Media Queries

Description:
User preference media queries detect user preferences set in their operating system, such as light or dark mode, and automatically adjust the website theme to match these preferences.
Example: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>User Preference Media Queries Example</title>
    <style>
        body {
            background-color: #ffffff;
            color: #000000;
        }

        @media (prefers-color-scheme: dark) {
            body {
                background-color: #333333;
                color: #ffffff;
            }
        }
    </style>
</head>
<body>
    <h2>User Preference Media Queries Example</h2>
    <p>This page adjusts based on your OS color scheme preference.</p>
</body>
</html>

3. CSS Variables for Theming

Description:
CSS variables (custom properties) allow for the definition of theme variables, making it easier to apply and switch themes. CSS variables enable centralized control over theme colors and other styles.
Example: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS Variables for Theming Example</title>
    <style>
        :root {
            --background-color: #ffffff;
            --text-color: #000000;
        }

        body {
            background-color: var(--background-color);
            color: var(--text-color);
        }

        .theme-toggle {
            margin: 20px;
            padding: 10px;
            cursor: pointer;
            background-color: #007BFF;
            color: white;
            border: none;
            border-radius: 5px;
        }

        .dark-theme {
            --background-color: #333333;
            --text-color: #ffffff;
        }
    </style>
</head>
<body>
    <h2>CSS Variables for Theming Example</h2>
    <button class="theme-toggle">Toggle Theme</button>

    <script>
        const toggleButton = document.querySelector('.theme-toggle');
        toggleButton.addEventListener('click', () => {
            document.body.classList.toggle('dark-theme');
        });
    </script>
</body>
</html>

4. Theme Switchers

Description:
Theme switchers provide users with a way to toggle between different themes, such as light and dark modes. This can be done using buttons or other UI elements that allow users to select their preferred theme.
Example: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Theme Switchers Example</title>
    <style>
        :root {
            --background-color: #ffffff;
            --text-color: #000000;
        }

        body {
            background-color: var(--background-color);
            color: var(--text-color);
        }

        .theme-toggle {
            margin: 20px;
            padding: 10px;
            cursor: pointer;
            background-color: #007BFF;
            color: white;
            border: none;
            border-radius: 5px;
        }

        .dark-theme {
            --background-color: #333333;
            --text-color: #ffffff;
        }
    </style>
</head>
<body>
    <h2>Theme Switchers Example</h2>
    <button class="theme-toggle" data-theme="light">Light Theme</button>
    <button class="theme-toggle" data-theme="dark">Dark Theme</button>

    <script>
        const themeButtons = document.querySelectorAll('.theme-toggle');
        themeButtons.forEach(button => {
            button.addEventListener('click', () => {
                document.body.className = button.getAttribute('data-theme') === 'dark' ? 'dark-theme' : '';
            });
        });
    </script>
</body>
</html>

5. Theming with Preprocessors

Description:
CSS preprocessors like Sass or Less provide advanced features for theming, such as variables, nested rules, and mixins. These features make it easier to manage and switch themes by defining and reusing styles efficiently.
Example (using Sass): 
// Define theme variables
$background-color-light: #ffffff;
$text-color-light: #000000;

$background-color-dark: #333333;
$text-color-dark: #ffffff;

// Mixin for theme
@mixin theme($bg-color, $text-color) {
    background-color: $bg-color;
    color: $text-color;
}

// Apply theme
body.light-theme {
    @include theme($background-color-light, $text-color-light);
}

body.dark-theme {
    @include theme($background-color-dark, $text-color-dark);
}

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Theming with Preprocessors Example</title>
    <link rel="stylesheet" href="styles.css"> <!-- Compiled CSS file from Sass -->
    <style>
        .theme-toggle {
            margin: 20px;
            padding: 10px;
            cursor: pointer;
            background-color: #007BFF;
            color: white;
            border: none;
            border-radius: 5px;
        }
    </style>
</head>
<body class="light-theme">
    <h2>Theming with Preprocessors Example</h2>
    <button class="theme-toggle">Toggle Theme</button>

    <script>
        const toggleButton = document.querySelector('.theme-toggle');
        toggleButton.addEventListener('click', () => {
            document.body.classList.toggle('dark-theme');
            document.body.classList.toggle('light-theme');
        });
    </script>
</body>
</html>

Explanation:

  1. Light and Dark Themes: Provides a way to switch between light and dark themes to improve readability and user comfort.
  2. User Preference Media Queries: Adjusts the theme based on user preferences set in their operating system.
  3. CSS Variables for Theming: Uses CSS variables to define and apply theme styles, making it easy to switch themes.
  4. Theme Switchers: Allows users to toggle between different themes using buttons or other UI elements.
  5. Theming with Preprocessors: Uses CSS preprocessors like Sass to define and manage themes more efficiently with variables and mixins.
These theming techniques enhance user experience by providing customizable and adaptable styles, ensuring that the website is comfortable and visually appealing for different users and environments.

Example Code

This example integrates all the theming and customization techniques: light and dark themes, user preference media queries, CSS variables for theming, theme switchers, and theming with preprocessors (using Sass).

HTML (index.html) 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Theming and Customization Example</title>
    <link rel="stylesheet" href="styles.css">
    <style>
        .theme-toggle {
            margin: 20px;
            padding: 10px;
            cursor: pointer;
            background-color: #007BFF;
            color: white;
            border: none;
            border-radius: 5px;
        }
    </style>
</head>
<body>
    <h2>Theming and Customization Example</h2>
    <p>This example integrates various theming techniques into a single page.</p>
    <button class="theme-toggle">Toggle Theme</button>

    <script>
        const toggleButton = document.querySelector('.theme-toggle');
        toggleButton.addEventListener('click', () => {
            if (document.body.classList.contains('dark-theme')) {
                document.body.classList.remove('dark-theme');
                document.body.classList.add('light-theme');
            } else {
                document.body.classList.remove('light-theme');
                document.body.classList.add('dark-theme');
            }
        });

        // Apply user preference for dark mode
        if (window.matchMedia && window.matchMedia('(prefers-color-scheme: dark)').matches) {
            document.body.classList.add('dark-theme');
        } else {
            document.body.classList.add('light-theme');
        }
    </script>
</body>
</html>

Sass (styles.scss)
