CSS Preprocessors

1. Introduction to SASS/SCSS

SASS (Syntactically Awesome Stylesheets) is a CSS preprocessor that allows for more dynamic and maintainable stylesheets. SCSS (Sassy CSS) is a newer syntax for SASS which is fully compatible with CSS.

2. Variables in SASS/SCSS

Variables allow you to store values and reuse them throughout your stylesheet.
Example Code: 
// Variables
$primary-color: #4CAF50;
$secondary-color: #ff9800;
$padding: 10px;

body {
    background-color: $primary-color;
    color: #fff;
    padding: $padding;
}

3. Nesting in SASS/SCSS

Nesting allows you to nest CSS selectors in a way that follows the same visual hierarchy of your HTML.
Example Code: 
nav {
    background-color: $secondary-color;
    
    ul {
        list-style: none;
        padding: 0;

        li {
            display: inline-block;
            margin-right: $padding;

            a {
                color: #fff;
                text-decoration: none;

                &:hover {
                    text-decoration: underline;
                }
            }
        }
    }
}

4. Mixins in SASS/SCSS

Mixins allow you to create reusable chunks of code that can be included in other selectors.
Example Code: 
@mixin border-radius($radius) {
    -webkit-border-radius: $radius;
       -moz-border-radius: $radius;
        -ms-border-radius: $radius;
         -o-border-radius: $radius;
            border-radius: $radius;
}

.box {
    @include border-radius(10px);
    background-color: $primary-color;
    color: #fff;
    padding: $padding;
}

5. Inheritance in SASS/SCSS

Inheritance allows you to share a set of CSS properties from one selector to another.
Example Code: 
// Base class
%message {
    padding: $padding;
    border: 1px solid #ccc;
    border-radius: 5px;
}

// Extending base class
.success {
    @extend %message;
    background-color: #dff0d8;
    border-color: #d6e9c6;
    color: #3c763d;
}

.error {
    @extend %message;
    background-color: #f2dede;
    border-color: #ebccd1;
    color: #a94442;
}

SCSS Code (styles.scss) 

// Variables
$primary-color: #4CAF50;
$secondary-color: #ff9800;
$success-color: #dff0d8;
$error-color: #f2dede;
$padding: 20px;
$border-radius: 10px;

// Mixin for border-radius
@mixin border-radius($radius) {
    -webkit-border-radius: $radius;
       -moz-border-radius: $radius;
            border-radius: $radius;
}

// Base class for messages
%message {
    padding: $padding;
    border: 1px solid #ccc;
    @include border-radius(5px);
    margin-bottom: $padding;
}

// Extendable message classes
.success {
    @extend %message;
    background-color: $success-color;
    border-color: darken($success-color, 10%);
    color: darken($success-color, 20%);
}

.error {
    @extend %message;
    background-color: $error-color;
    border-color: darken($error-color, 10%);
    color: darken($error-color, 20%);
}

// Styles for the box
.box {
    padding: $padding;
    @include border-radius($border-radius);
    background-color: $primary-color;
    color: white;
    margin-bottom: $padding;

    nav {
        ul {
            list-style: none;
            padding: 0;

            li {
                display: inline-block;
                margin-right: 10px;

                a {
                    color: white;
                    text-decoration: none;

                    &:hover {
                        text-decoration: underline;
                    }
                }
            }
        }
    }
}
