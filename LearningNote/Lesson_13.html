Transformations

1. translate

The translate() function moves an element from its current position along the X and Y axes. 
.translate-example {
    transform: translate(50px, 20px);
}

2. rotate

The rotate() function rotates an element clockwise around a fixed point. 
.rotate-example {
    transform: rotate(45deg);
}

3. scale

The scale() function scales an element by increasing or decreasing its size. 
.scale-example {
    transform: scale(1.5);
}

4. skew

The skew() function skews an element along the X and Y axes by a given angle. 
.skew-example {
    transform: skew(20deg, 10deg);
}

5. matrix

The matrix() function provides a 2D transformation using a combination of translation, rotation, scale, and skew. 
.matrix-example {
    transform: matrix(1, 0.5, -0.5, 1, 30, 10);
}

Example Code 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Interactive CSS Transformations</title>
    <style>
        .container {
            display: flex;
            flex-wrap: wrap;
            gap: 20px;
            padding: 20px;
        }

        .box {
            width: 100px;
            height: 100px;
            background-color: red;
            transition: transform 0.5s ease;
        }

        .buttons {
            margin: 20px 0;
        }

        button {
            margin: 5px;
            padding: 10px 20px;
            cursor: pointer;
        }
    </style>
</head>
<body>
    <div class="container">
        <div class="box" id="box1"></div>
        <div class="box" id="box2"></div>
        <div class="box" id="box3"></div>
        <div class="box" id="box4"></div>
        <div class="box" id="box5"></div>
    </div>

    <div class="buttons">
        <button onclick="applyTransform('translate')">Translate</button>
        <button onclick="applyTransform('rotate')">Rotate</button>
        <button onclick="applyTransform('scale')">Scale</button>
        <button onclick="applyTransform('skew')">Skew</button>
        <button onclick="applyTransform('matrix')">Matrix</button>
        <button onclick="resetTransform()">Reset</button>
    </div>

    <script>
        function applyTransform(transformType) {
            resetTransform();
            const box1 = document.getElementById('box1');
            const box2 = document.getElementById('box2');
            const box3 = document.getElementById('box3');
            const box4 = document.getElementById('box4');
            const box5 = document.getElementById('box5');

            switch(transformType) {
                case 'translate':
                    box1.style.transform = 'translate(50px, 100px)';
                    break;
                case 'rotate':
                    box2.style.transform = 'rotate(45deg)';
                    break;
                case 'scale':
                    box3.style.transform = 'scale(1.5, 2)';
                    break;
                case 'skew':
                    box4.style.transform = 'skew(20deg, 10deg)';
                    break;
                case 'matrix':
                    box5.style.transform = 'matrix(1, 0.5, -0.5, 1, 50, 100)';
                    break;
                default:
                    break;
            }
        }

        function resetTransform() {
            const boxes = document.querySelectorAll('.box');
            boxes.forEach(box => {
                box.style.transform = 'none';
            });
        }
    </script>
</body>
</html>
