Advanced Responsive Design

1. Flexbox for Responsive Design

Flexbox is a layout model that allows you to design flexible and responsive layouts. It works well for both simple and complex layouts.
  • Basic Example: 
    <div class="flex-container">
    <div class="flex-item">Item 1</div>
    <div class="flex-item">Item 2</div>
    <div class="flex-item">Item 3</div>
</div>

    .flex-container {
    display: flex;
    flex-wrap: wrap;
    gap: 10px;
}

.flex-item {
    flex: 1 1 200px; /* Grow, shrink, basis */
    padding: 20px;
    background-color: lightcoral;
}

2. Grid for Responsive Design

CSS Grid is a powerful layout system that allows you to create complex, responsive layouts with ease.
  • Basic Example: 
    <div class="grid-container">
    <div class="grid-item">Item 1</div>
    <div class="grid-item">Item 2</div>
    <div class="grid-item">Item 3</div>
    <div class="grid-item">Item 4</div>
</div>

    .grid-container {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
    gap: 10px;
}

.grid-item {
    padding: 20px;
    background-color: lightblue;
}

3. CSS Frameworks (Bootstrap, Foundation)

CSS frameworks like Bootstrap and Foundation provide pre-written CSS and JavaScript to create responsive web designs quickly.
  • Bootstrap Example:
    <link href="https://stackpath.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css" rel="stylesheet">
<div class="container">
    <div class="row">
        <div class="col-sm">Column 1</div>
        <div class="col-sm">Column 2</div>
        <div class="col-sm">Column 3</div>
    </div>
</div>
  • Foundation Example:
    <link href="https://cdnjs.cloudflare.com/ajax/libs/foundation/6.6.3/css/foundation.min.css" rel="stylesheet">
<div class="grid-container">
    <div class="grid-x grid-padding-x">
        <div class="cell small-4">Column 1</div>
        <div class="cell small-4">Column 2</div>
        <div class="cell small-4">Column 3</div>
    </div>
</div>

4. Responsive Images

Responsive images ensure that images are scaled appropriately for different screen sizes and resolutions.
  • <img> with srcset: 
    <img src="small.jpg" 
     srcset="medium.jpg 600w, large.jpg 1200w" 
     sizes="(max-width: 600px) 100vw, 50vw" 
     alt="Responsive Image">
  • CSS Background Images:
    <div class="responsive-background"></div>

    .responsive-background {
    width: 100%;
    height: 300px;
    background-image: url('small.jpg');
    background-size: cover;
    background-position: center;
}

@media (min-width: 600px) {
    .responsive-background {
        background-image: url('medium.jpg');
    }
}

@media (min-width: 1200px) {
    .responsive-background {
        background-image: url('large.jpg');
    }
}

5. CSS Variables (Custom Properties)

CSS variables allow you to define reusable values throughout your CSS, making it easier to maintain and update styles.
  • Example: 
    <div class="variable-container">
    Variable-based Design
</div>

    :root {
    --primary-color: lightcoral;
    --secondary-color: lightblue;
    --padding: 20px;
}

.variable-container {
    background-color: var(--primary-color);
    color: var(--secondary-color);
    padding: var(--padding);
}

@media (min-width: 600px) {
    :root {
        --primary-color: lightblue;
        --secondary-color: lightcoral;
    }
}
