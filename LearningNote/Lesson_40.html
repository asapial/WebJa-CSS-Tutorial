CSS for Print

1. Page Breaks

Page breaks control how content is split across printed pages. They ensure content like headings, tables, and images are not split awkwardly.

Key Points:

  • Use page-break-before, page-break-after, and page-break-inside to control page breaks.
  • Common values include auto, always, avoid, left, and right.

Example: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page Breaks</title>
    <style>
        @media print {
            h1 {
                page-break-before: always;
            }

            .no-break {
                page-break-inside: avoid;
            }
        }
    </style>
</head>
<body>
    <h1>Chapter 1</h1>
    <p class="no-break">Content that should not break.</p>
    <h1>Chapter 2</h1>
    <p class="no-break">More content that should not break.</p>
</body>
</html>

2. Styling for Print Media

Styling for print media involves creating styles that ensure content looks good when printed. You can hide unnecessary elements, change fonts, and adjust colors.

Key Points:

  • Use @media print to apply styles specifically for print.
  • Hide navigation, ads, and other non-essential elements.

Example: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Styling for Print Media</title>
    <style>
        @media print {
            body {
                font-size: 12pt;
                line-height: 1.4;
            }

            .no-print {
                display: none;
            }

            .print-only {
                display: block;
            }
        }

        .print-only {
            display: none;
        }
    </style>
</head>
<body>
    <div class="no-print">This will not be printed.</div>
    <div class="print-only">This will only be printed.</div>
    <p>Printable content.</p>
</body>
</html>

3. @page Rule

The @page rule is used to modify the dimensions and margins of the printed page. It can also add content to page margins.

Key Points:

  • Use @page to set page size, margins, and orientation.
  • Use @top-left, @top-right, @bottom-left, and @bottom-right to add headers and footers.

Example: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>@page Rule</title>
    <style>
        @page {
            size: A4;
            margin: 1in;
        }

        @page :first {
            margin-top: 2in;
        }

        @page {
            @top-right {
                content: "Page " counter(page);
            }
        }
    </style>
</head>
<body>
    <h1>Document Title</h1>
    <p>Content of the document.</p>
</body>
</html>

4. CSS Units for Print

Print styles often use absolute units like in (inches), cm (centimeters), mm (millimeters), and pt (points) to ensure consistent measurements.

Key Points:

  • Use absolute units for precise control over printed dimensions.
  • pt (points) is commonly used for font sizes in print.

Example: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS Units for Print</title>
    <style>
        @media print {
            body {
                font-size: 10pt;
                margin: 1in;
            }

            .box {
                width: 5cm;
                height: 3cm;
                border: 1mm solid black;
            }
        }
    </style>
</head>
<body>
    <div class="box">A box with print-specific dimensions.</div>
</body>
</html>

5. Print-specific Styles

Print-specific styles ensure that your content is optimized for printing, such as removing navigation and adjusting colors.

Key Points:

  • Use @media print to apply styles specifically for print.
  • Optimize layout and content visibility for the printed page.

Example: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Print-specific Styles</title>
    <style>
        @media print {
            body {
                background: white;
                color: black;
                font-size: 12pt;
            }

            nav, .advertisement, .footer {
                display: none;
            }

            .content {
                width: 100%;
            }
        }

        .advertisement {
            background: yellow;
            padding: 20px;
            margin: 10px 0;
        }
    </style>
</head>
<body>
    <nav>Navigation Menu</nav>
    <div class="advertisement">Advertisement</div>
    <div class="content">
        <h1>Printable Content</h1>
        <p>This is the main content of the page.</p>
    </div>
    <footer class="footer">Footer Information</footer>
</body>
</html>

Example Code

Here’s a combined example that incorporates all the print-specific techniques discussed: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Print Styles</title>
    <style>
        @media print {
            body {
                font-size: 12pt;
                line-height: 1.5;
                margin: 1in;
                background: white;
                color: black;
            }

            nav, .advertisement, .footer {
                display: none;
            }

            h1 {
                page-break-before: always;
            }

            .no-break {
                page-break-inside: avoid;
            }

            .print-only {
                display: block;
            }

            .content {
                width: 100%;
            }
        }

        @page {
            size: A4;
            margin: 1in;

            @top-right {
                content: "Page " counter(page);
            }
        }

        .print-only {
            display: none;
        }
    </style>
</head>
<body>
    <nav>Navigation Menu</nav>
    <div class="advertisement">Advertisement</div>
    <div class="content">
        <h1>Chapter 1</h1>
        <p class="no-break">Content that should not break.</p>
        <h1>Chapter 2</h1>
        <p class="no-break">More content that should not break.</p>
    </div>
    <footer class="footer">Footer Information</footer>
</body>
</html>
