CSS and JavaScript Integration

In this lesson, we will explore how to integrate CSS with JavaScript to create dynamic and interactive web experiences. We will cover adding and removing classes, manipulating styles directly with JavaScript, using CSS transitions with JavaScript, interacting with CSS variables, and using the Intersection Observer API.

1. Adding and Removing Classes

Description:
Adding and removing classes with JavaScript allows you to dynamically change the styles of elements based on user interactions or other events.
Example: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Adding and Removing Classes</title>
    <style>
        .box {
            width: 100px;
            height: 100px;
            background-color: lightblue;
            margin: 20px;
            transition: background-color 0.3s;
        }

        .active {
            background-color: lightcoral;
        }
    </style>
</head>
<body>
    <div class="box"></div>
    <button id="toggleClass">Toggle Class</button>

    <script>
        const box = document.querySelector('.box');
        const button = document.getElementById('toggleClass');

        button.addEventListener('click', () => {
            box.classList.toggle('active');
        });
    </script>
</body>
</html>

2. Manipulating Styles with JavaScript

Description:
You can directly manipulate the styles of elements using JavaScript to dynamically adjust their appearance.
Example: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Manipulating Styles with JavaScript</title>
    <style>
        .box {
            width: 100px;
            height: 100px;
            background-color: lightblue;
            margin: 20px;
            transition: transform 0.3s, background-color 0.3s;
        }
    </style>
</head>
<body>
    <div class="box"></div>
    <button id="changeStyle">Change Style</button>

    <script>
        const box = document.querySelector('.box');
        const button = document.getElementById('changeStyle');

        button.addEventListener('click', () => {
            box.style.transform = 'scale(1.5)';
            box.style.backgroundColor = 'lightcoral';
        });
    </script>
</body>
</html>

4. CSS Variables and JavaScript

Description:
CSS variables (custom properties) can be manipulated with JavaScript to dynamically change the values of styles throughout the document.
Example: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS Variables and JavaScript</title>
    <style>
        :root {
            --box-color: lightblue;
            --box-size: 100px;
        }

        .box {
            width: var(--box-size);
            height: var(--box-size);
            background-color: var(--box-color);
            margin: 20px;
            transition: background-color 0.3s, width 0.3s, height 0.3s;
        }
    </style>
</head>
<body>
    <div class="box"></div>
    <button id="changeVariables">Change Variables</button>

    <script>
        const root = document.documentElement;
        const button = document.getElementById('changeVariables');

        button.addEventListener('click', () => {
            root.style.setProperty('--box-color', 'lightcoral');
            root.style.setProperty('--box-size', '150px');
        });
    </script>
</body>
</html>

5. Intersection Observer API

Description:
The Intersection Observer API allows you to observe changes in the intersection of a target element with an ancestor element or the viewport. It is useful for lazy-loading images, triggering animations, and more.
Example: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Intersection Observer API</title>
    <style>
        .box {
            width: 100%;
            height: 300px;
            margin: 50px 0;
            background-color: lightblue;
            transition: transform 0.5s, opacity 0.5s;
            opacity: 0;
            transform: translateY(100px);
        }

        .box.visible {
            opacity: 1;
            transform: translateY(0);
        }
    </style>
</head>
<body>
    <div class="box"></div>
    <div class="box"></div>
    <div class="box"></div>

    <script>
        const boxes = document.querySelectorAll('.box');

        const observer = new IntersectionObserver(entries => {
            entries.forEach(entry => {
                if (entry.isIntersecting) {
                    entry.target.classList.add('visible');
                }
            });
        }, {
            threshold: 0.5
        });

        boxes.forEach(box => {
            observer.observe(box);
        });
    </script>
</body>
</html>
