Colors and Gradients

1. Hexadecimal Colors

Hexadecimal colors are specified using the #RRGGBB format, where RR, GG, and BB are hexadecimal values (00 to FF) representing the red, green, and blue components.
  • Syntax:
    color: #3498db;
  • Example:
    <h2>Hexadecimal Colors</h2>
<div class="hex-color">This text uses a hexadecimal color.</div>

    .hex-color {
    color: #3498db; /* Blue */
    background-color: #ecf0f1; /* Light gray */
    padding: 10px;
}

2. RGB and RGBA Colors

RGB colors are specified using the rgb() function, which takes three arguments: red, green, and blue. RGBA is similar but includes an alpha (opacity) component.
  • Syntax:
    color: rgb(52, 152, 219);
color: rgba(52, 152, 219, 0.5);
  • Example:
    <h2>RGB and RGBA Colors</h2>
<div class="rgb-color">This text uses an RGB color.</div>
<div class="rgba-color">This text uses an RGBA color.</div>

    .rgb-color {
    color: rgb(52, 152, 219); /* Blue */
    background-color: rgb(236, 240, 241); /* Light gray */
    padding: 10px;
}

.rgba-color {
    color: rgba(52, 152, 219, 0.5); /* Semi-transparent blue */
    background-color: rgba(236, 240, 241, 0.5); /* Semi-transparent light gray */
    padding: 10px;
}

3. HSL and HSLA Colors

HSL colors are specified using the hsl() function, which stands for hue, saturation, and lightness. HSLA is similar but includes an alpha (opacity) component.
  • Syntax:
    color: hsl(204, 70%, 53%);
color: hsla(204, 70%, 53%, 0.5);
  • Example:
    <h2>HSL and HSLA Colors</h2>
<div class="hsl-color">This text uses an HSL color.</div>
<div class="hsla-color">This text uses an HSLA color.</div>

    .hsl-color {
    color: hsl(204, 70%, 53%); /* Blue */
    background-color: hsl(0, 0%, 92%); /* Light gray */
    padding: 10px;
}

.hsla-color {
    color: hsla(204, 70%, 53%, 0.5); /* Semi-transparent blue */
    background-color: hsla(0, 0%, 92%, 0.5); /* Semi-transparent light gray */
    padding: 10px;
}

4. Linear Gradients

Linear gradients are specified using the linear-gradient() function, which creates a gradient along a straight line.
  • Syntax:
    background: linear-gradient(to right, #3498db, #2ecc71);
  • Example:
    <h2>Linear Gradients</h2>
<div class="linear-gradient">This box uses a linear gradient.</div>

    .linear-gradient {
    background: linear-gradient(to right, #3498db, #2ecc71); /* Blue to green */
    padding: 20px;
    color: white;
    text-align: center;
}

5. Radial Gradients

Radial gradients are specified using the radial-gradient() function, which creates a gradient radiating from an origin.
  • Syntax:
    background: radial-gradient(circle, #3498db, #2ecc71);
  • Example:
    <h2>Radial Gradients</h2>
<div class="radial-gradient">This box uses a radial gradient.</div>

    .radial-gradient {
    background: radial-gradient(circle, #3498db, #2ecc71); /* Blue to green */
    padding: 20px;
    color: white;
    text-align: center;
}
