Advanced CSS Layout Techniques

1. Holy Grail Layout

Description:
The Holy Grail Layout is a classic web design pattern that consists of a header, footer, three columns (with the main content in the center column), and flexible layout. This layout has been a challenge due to its need for flexible and stable column sizes and the full-width header and footer.
Example: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Holy Grail Layout</title>
    <style>
        body {
            display: grid;
            grid-template-areas:
                "header header header"
                "nav content ads"
                "footer footer footer";
            grid-template-rows: auto 1fr auto;
            grid-template-columns: 200px 1fr 200px;
            height: 100vh;
            margin: 0;
        }
        header {
            grid-area: header;
            background: lightblue;
            padding: 1em;
        }
        nav {
            grid-area: nav;
            background: lightcoral;
            padding: 1em;
        }
        main {
            grid-area: content;
            background: lightgoldenrodyellow;
            padding: 1em;
        }
        aside {
            grid-area: ads;
            background: lightgreen;
            padding: 1em;
        }
        footer {
            grid-area: footer;
            background: lightgray;
            padding: 1em;
        }
    </style>
</head>
<body>
    <header>Header</header>
    <nav>Navigation</nav>
    <main>Main Content</main>
    <aside>Advertisements</aside>
    <footer>Footer</footer>
</body>
</html>

2. CSS Grid Masonry Layout

Description:
CSS Grid Masonry Layout is used to create a Pinterest-like layout with uneven column heights, achieved using CSS Grid and properties like grid-auto-rows and grid-row.
Example: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS Grid Masonry Layout</title>
    <style>
        .grid {
            display: grid;
            grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
            grid-auto-rows: 10px;
            gap: 10px;
        }
        .grid-item {
            background: lightblue;
            margin-bottom: 10px;
            padding: 10px;
            border: 1px solid #ccc;
        }
        .grid-item:nth-child(odd) {
            grid-row: span 2;
        }
    </style>
</head>
<body>
    <div class="grid">
        <div class="grid-item">Item 1</div>
        <div class="grid-item">Item 2</div>
        <div class="grid-item">Item 3</div>
        <div class="grid-item">Item 4</div>
        <div class="grid-item">Item 5</div>
        <div class="grid-item">Item 6</div>
    </div>
</body>
</html>

3. Flexbox Masonry Layout

Description:
Flexbox Masonry Layout is another approach to creating a Pinterest-like layout using Flexbox. It involves using flex and order properties to achieve the masonry effect.
Example: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Flexbox Masonry Layout</title>
    <style>
        .container {
            display: flex;
            flex-wrap: wrap;
            gap: 10px;
        }
        .item {
            flex: 1 1 200px;
            background: lightcoral;
            margin: 10px;
            padding: 10px;
            border: 1px solid #ccc;
        }
        .item:nth-child(odd) {
            flex: 2 1 200px;
        }
    </style>
</head>
<body>
    <div class="container">
        <div class="item">Item 1</div>
        <div class="item">Item 2</div>
        <div class="item">Item 3</div>
        <div class="item">Item 4</div>
        <div class="item">Item 5</div>
        <div class="item">Item 6</div>
    </div>
</body>
</html>

4. CSS Grid with Flexbox

Description:
Combining CSS Grid and Flexbox leverages the strengths of both layout techniques. CSS Grid is used for the overall layout structure, while Flexbox is used for aligning items within the grid cells.
Example: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS Grid with Flexbox</title>
    <style>
        .grid {
            display: grid;
            grid-template-columns: repeat(3, 1fr);
            gap: 20px;
        }
        .grid-item {
            background: lightblue;
            display: flex;
            align-items: center;
            justify-content: center;
            padding: 20px;
            border: 1px solid #ccc;
        }
    </style>
</head>
<body>
    <div class="grid">
        <div class="grid-item">Item 1</div>
        <div class="grid-item">Item 2</div>
        <div class="grid-item">Item 3</div>
        <div class="grid-item">Item 4</div>
        <div class="grid-item">Item 5</div>
        <div class="grid-item">Item 6</div>
    </div>
</body>
</html>

5. Complex Responsive Layouts

Description:
Complex responsive layouts use a combination of CSS Grid and Flexbox, media queries, and other CSS techniques to create layouts that adapt to different screen sizes and orientations.
Example: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Complex Responsive Layout</title>
    <style>
        .container {
            display: grid;
            grid-template-areas:
                "header header header"
                "nav content ads"
                "footer footer footer";
            grid-template-rows: auto 1fr auto;
            grid-template-columns: 200px 1fr 200px;
            height: 100vh;
            gap: 10px;
        }
        header {
            grid-area: header;
            background: lightblue;
            padding: 1em;
        }
        nav {
            grid-area: nav;
            background: lightcoral;
            padding: 1em;
        }
        main {
            grid-area: content;
            background: lightgoldenrodyellow;
            padding: 1em;
            display: flex;
            flex-direction: column;
            justify-content: center;
            align-items: center;
        }
        aside {
            grid-area: ads;
            background: lightgreen;
            padding: 1em;
        }
        footer {
            grid-area: footer;
            background: lightgray;
            padding: 1em;
        }
        @media (max-width: 768px) {
            .container {
                grid-template-areas:
                    "header"
                    "nav"
                    "content"
                    "ads"
                    "footer";
                grid-template-columns: 1fr;
                grid-template-rows: auto;
            }
        }
    </style>
</head>
<body>
    <div class="container">
        <header>Header</header>
        <nav>Navigation</nav>
        <main>Main Content</main>
        <aside>Advertisements</aside>
        <footer>Footer</footer>
    </div>
</body>
</html>
