Transitions and Animations

1. CSS Transitions

CSS transitions provide a way to animate changes to CSS properties smoothly over a specified duration. They allow for changes in the state of an element to occur over a period, rather than instantly, providing a more engaging and dynamic user experience.

Key Points:

  • Transitions are triggered by changes in CSS property values.
  • They occur when an element changes from one state to another.
  • Useful for enhancing user interactions and creating visual effects.

Example: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <style>
        .box {
            width: 100px;
            height: 100px;
            background-color: blue;
            transition: background-color 0.5s, transform 0.5s;
        }

        .box:hover {
            background-color: red;
            transform: rotate(45deg);
        }
    </style>
</head>
<body>
    <div class="box"></div>
</body>
</html>

2. transition-property

The transition-property specifies the CSS properties to which a transition effect should be applied. Multiple properties can be listed, separated by commas.

Key Points:

  • Defines the properties that will be animated.
  • Can specify one or multiple properties.
  • Defaults to all if not specified, which means all properties will be transitioned.

Example: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <style>
        .box {
            width: 100px;
            height: 100px;
            background-color: blue;
            transition-property: background-color, width;
            transition-duration: 0.5s;
        }

        .box:hover {
            background-color: red;
            width: 200px;
        }
    </style>
</head>
<body>
    <div class="box"></div>
</body>
</html>

3. transition-duration

The transition-duration specifies the duration over which a transition effect should occur. It is defined in seconds (s) or milliseconds (ms).

Key Points:

  • Specifies how long the transition will take.
  • Can define different durations for different properties.
  • Defaults to 0s if not specified, meaning no transition.

Example: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <style>
        .box {
            width: 100px;
            height: 100px;
            background-color: blue;
            transition-property: background-color, width;
            transition-duration: 1s, 0.5s;
        }

        .box:hover {
            background-color: red;
            width: 200px;
        }
    </style>
</head>
<body>
    <div class="box"></div>
</body>
</html>

4. CSS Animations

CSS animations allow more complex and controlled animations than transitions. They are defined using the @keyframes rule, which specifies the intermediate steps in an animation sequence.

Key Points:

  • Allows for more complex and continuous animations.
  • Defined using @keyframes for specifying keyframes and animation steps.
  • Can control the timing, duration, and iteration of the animation.

Example: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <style>
        @keyframes move {
            from {
                transform: translateX(0);
            }
            to {
                transform: translateX(100px);
            }
        }

        .box {
            width: 100px;
            height: 100px;
            background-color: blue;
            animation: move 2s infinite alternate;
        }
    </style>
</head>
<body>
    <div class="box"></div>
</body>
</html>

5. @keyframes

The @keyframes rule is used to define the intermediate steps of a CSS animation. Each keyframe within @keyframes defines the styles for a specific moment in the animation sequence.

Key Points:

  • Defines the keyframes for an animation.
  • Uses percentages or keywords (from, to) to specify keyframes.
  • Can include any number of keyframes with varying styles.

Example: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <style>
        @keyframes colorChange {
            0% {
                background-color: blue;
            }
            50% {
                background-color: green;
            }
            100% {
                background-color: red;
            }
        }

        .box {
            width: 100px;
            height: 100px;
            animation: colorChange 3s infinite;
        }
    </style>
</head>
<body>
    <div class="box"></div>
</body>
</html>

Example Code

Here is an example that incorporates CSS transitions and animations, along with the use of transition-property, transition-duration, and @keyframes: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <style>
        /* Transition for the box */
        .box {
            width: 100px;
            height: 100px;
            margin-top: 10%;
            margin-left: 5%;
            background-color: blue;
            transition-property: background-color, transform;
            transition-duration: 0.5s, 1s;
        }

        .box:hover {
            background-color: red;
            transform: rotate(45deg);
        }

        /* Animation for the circle */
        @keyframes moveAndColor {
            0% {
                transform: translateX(0);
                background-color: yellow;
            }
            50% {
                transform: translateX(100px);
                background-color: green;
            }
            100% {
                transform: translateX(0);
                background-color: yellow;
            }
        }

        .circle {
            width: 100px;
            height: 100px;
            background-color: yellow;
            border-radius: 50%;
            animation: moveAndColor 2s infinite;
        }
    </style>
</head>
<body>
    <div class="box"></div>
    <br><br>
    <div class="circle"></div>
</body>
</html>
