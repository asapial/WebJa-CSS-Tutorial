Advanced Text Effects

1. Text Shadows

Description:
Text shadows add depth and dimension to text, making it stand out. The text-shadow property allows you to create one or more shadows on text.
Example: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Text Shadows</title>
    <style>
        .shadow {
            font-size: 2em;
            color: #333;
            text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.5);
        }
    </style>
</head>
<body>
    <p class="shadow">This is text with a shadow effect.</p>
</body>
</html>

2. Text Gradients

Description:
Text gradients use background images with the background-clip property set to text, and the text color set to transparent. This creates a gradient effect on the text.
Example: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Text Gradients</title>
    <style>
        .gradient {
            font-size: 2em;
            font-weight: bold;
            background: linear-gradient(to right, #ff7e5f, #feb47b);
            -webkit-background-clip: text;
            color: transparent;
        }
    </style>
</head>
<body>
    <p class="gradient">This is text with a gradient effect.</p>
</body>
</html>

3. Text Stroke

Description:
Text stroke adds an outline to text, which can create a striking visual effect. The -webkit-text-stroke property is used for this purpose, though it's only supported in WebKit browsers.
Example: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Text Stroke</title>
    <style>
        .stroke {
            font-size: 2em;
            font-weight: bold;
            color: white;
            -webkit-text-stroke: 2px black;
        }
    </style>
</head>
<body>
    <p class="stroke">This is text with a stroke effect.</p>
</body>
</html>

4. Letter Spacing and Line Height

Description:
Letter spacing adjusts the space between characters in text, while line height adjusts the space between lines of text. These properties can improve readability and visual appearance.
Example: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Letter Spacing and Line Height</title>
    <style>
        .spacing {
            font-size: 1.5em;
            letter-spacing: 0.1em;
            line-height: 1.6;
        }
    </style>
</head>
<body>
    <p class="spacing">This is text with adjusted letter spacing and line height for improved readability.</p>
</body>
</html>

5. Text Transformations

Description:
Text transformations change the capitalization of text. The text-transform property can be used to make text uppercase, lowercase, or capitalize each word.
Example: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Text Transformations</title>
    <style>
        .uppercase {
            text-transform: uppercase;
        }
        .lowercase {
            text-transform: lowercase;
        }
        .capitalize {
            text-transform: capitalize;
        }
    </style>
</head>
<body>
    <p class="uppercase">This is uppercase text.</p>
    <p class="lowercase">This is lowercase text.</p>
    <p class="capitalize">this is capitalized text.</p>
</body>
</html>

Example:

        }

            text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.5);
        }
        .text-effects .gradient {
            font-size: 2em;
            font-weight: bold;
            background: linear-gradient(to right, #ff7e5f, #feb47b);
            -webkit-background-clip: text;
            color: transparent;
        }
        .text-effects .stroke {
            font-size: 2em;
            font-weight: bold;
            color: white;
            -webkit-text-stroke: 2px black;
        }
        .text-effects .spacing {
            font-size: 1.5em;
            letter-spacing: 0.1em;
            line-height: 1.6;
        }
        .text-effects .uppercase {
            text-transform: uppercase;
        }
        .text-effects .lowercase {

</head>
