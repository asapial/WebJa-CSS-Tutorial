Performance Optimization

1. CSS Minification

CSS minification is the process of removing unnecessary characters from the CSS code, such as white spaces, comments, and line breaks, without affecting its functionality. This reduces the file size and improves load times.

Key Points:

  • Use tools like CSSNano, CleanCSS, or online minification services.
  • Integrate minification into build processes using task runners like Gulp or Grunt.

Example:

Original CSS: 
/* Original CSS */
body {
    background-color: #ffffff;
    margin: 0;
    padding: 0;
}
h1 {
    color: #333333;
    font-size: 24px;
    margin-bottom: 10px;
}
Minified CSS: 
body{background-color:#fff;margin:0;padding:0}h1{color:#333;font-size:24px;margin-bottom:10px}

2. Critical CSS

Critical CSS refers to the CSS needed to render the above-the-fold content of a webpage. By inlining critical CSS, you can speed up the rendering of your web page, reducing the time it takes for users to see the content.

Key Points:

  • Identify the CSS required for initial rendering.
  • Inline critical CSS directly in the HTML <head>.

Example: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Critical CSS Example</title>
    <style>
        /* Critical CSS inlined */
        body {
            background-color: #ffffff;
            margin: 0;
            padding: 0;
        }
        h1 {
            color: #333333;
            font-size: 24px;
            margin-bottom: 10px;
        }
    </style>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <h1>Welcome to My Website</h1>
    <p>Content of the webpage...</p>
</body>
</html>

3. CSS Compression

CSS compression reduces the size of CSS files by encoding them efficiently. Gzip and Brotli are commonly used compression algorithms for web assets.

Key Points:

  • Enable compression on your web server (Apache, Nginx).
  • Use tools or plugins to compress CSS files.

Example:

Nginx Configuration: 
server {
    listen 80;
    server_name example.com;

    location / {
        root /var/www/html;
        index index.html;
    }

    # Enable Gzip compression
    gzip on;
    gzip_types text/css application/javascript;
    gzip_proxied any;
    gzip_min_length 256;
}

4. Reducing Repaints and Reflows

Repaints and reflows are browser rendering processes that can impact performance. Minimizing these processes can lead to a smoother user experience.

Key Points:

  • Avoid inline styles and use class-based styles instead.
  • Minimize DOM changes and complex CSS selectors.
  • Use CSS properties that do not trigger reflow (e.g., transform instead of top/left).

Example: 

/* Efficient styles */
.hidden {
    display: none;
}

.transformed {
    transform: translateX(10px);
}

/* Inefficient styles */
#content div:first-child {
    margin-top: 10px;
}

5. Efficient CSS Selectors

Efficient CSS selectors improve rendering performance by reducing the complexity of the selection process.

Key Points:

  • Use simple selectors like class and ID selectors.
  • Avoid universal and descendant selectors.
  • Limit the use of attribute selectors.

Example: 

/* Efficient selectors */
.button {
    background-color: blue;
    color: white;
}

/* Inefficient selectors */
div > p > a[href*="example"] {
    color: red;
}

Example Code

Here’s a combined example that incorporates all the performance optimization techniques discussed: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Performance Optimization Example</title>
    <!-- Inlining Critical CSS -->
    <style>
        body {
            background-color: #ffffff;
            margin: 0;
            padding: 0;
        }
        h1 {
            color: #333333;
            font-size: 24px;
            margin-bottom: 10px;
        }
        .button {
            background-color: blue;
            color: white;
            padding: 10px 20px;
            border: none;
            cursor: pointer;
            transition: transform 0.3s;
        }
        .button:hover {
            transform: scale(1.1);
        }
    </style>
    <link rel="stylesheet" href="styles.min.css">
</head>
<body>
    <h1>Welcome to My Optimized Website</h1>
    <p class="no-break">This is a sample content that demonstrates various performance optimization techniques in CSS.</p>
    <button class="button">Click Me</button>

    <!-- Example of Efficient CSS Selector -->
    <div id="content">
        <div class="inner-content">
            <p>Inner content with efficient CSS selectors.</p>
        </div>
    </div>
</body>
</html>
