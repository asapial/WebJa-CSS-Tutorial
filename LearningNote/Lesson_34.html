CSS Custom Properties

1. Using CSS Variables

CSS Variables, also known as Custom Properties, allow you to define reusable values throughout your stylesheet. They are defined using the -- prefix and can be accessed using the var() function.

Key Points:

  • Define a variable: --variable-name: value;
  • Use a variable: property: var(--variable-name);

Example: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS Variables Example</title>
    <style>
        :root {
            --primary-color: #3498db;
            --secondary-color: #2ecc71;
            --padding: 10px;
        }

        .box {
            background-color: var(--primary-color);
            padding: var(--padding);
            color: white;
            margin: 10px;
        }

        .box-secondary {
            background-color: var(--secondary-color);
        }
    </style>
</head>
<body>
    <div class="box">Primary Box</div>
    <div class="box box-secondary">Secondary Box</div>
</body>
</html>

2. Updating Variables with JavaScript

You can update CSS Variables dynamically using JavaScript. This allows for real-time updates and interactivity.

Example: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Update CSS Variables with JavaScript</title>
    <style>
        :root {
            --primary-color: #3498db;
            --padding: 10px;
        }

        .box {
            background-color: var(--primary-color);
            padding: var(--padding);
            color: white;
            margin: 10px;
        }
    </style>
</head>
<body>
    <div class="box">Primary Box</div>
    <button id="changeColor">Change Color</button>

    <script>
        document.getElementById('changeColor').addEventListener('click', function() {
            document.documentElement.style.setProperty('--primary-color', '#e74c3c');
        });
    </script>
</body>
</html>

3. Local Scope vs Global Scope

CSS Variables can be defined at the global level (using :root) or at the local level (within a specific selector). Local variables override global ones within their scope.

Example: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Local vs Global Scope</title>
    <style>
        :root {
            --primary-color: #3498db;
        }

        .box {
            background-color: var(--primary-color);
            color: white;
            padding: 10px;
            margin: 10px;
        }

        .box-local {
            --primary-color: #e74c3c;
            background-color: var(--primary-color);
        }
    </style>
</head>
<body>
    <div class="box">Global Variable</div>
    <div class="box box-local">Local Variable</div>
</body>
</html>

4. Dynamic Theming with Variables

CSS Variables are great for dynamic theming, allowing you to easily switch themes by updating a few variables.

Example: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Dynamic Theming</title>
    <style>
        :root {
            --background-color: #f0f0f0;
            --text-color: #333;
        }

        .dark-theme {
            --background-color: #333;
            --text-color: #f0f0f0;
        }

        body {
            background-color: var(--background-color);
            color: var(--text-color);
            font-family: Arial, sans-serif;
            padding: 20px;
        }
    </style>
</head>
<body>
    <button id="toggleTheme">Toggle Theme</button>
    <p>This is a paragraph demonstrating dynamic theming.</p>

    <script>
        document.getElementById('toggleTheme').addEventListener('click', function() {
            document.body.classList.toggle('dark-theme');
        });
    </script>
</body>
</html>

5. Fallback Values in Variables

CSS Variables can have fallback values in case the variable is not defined or fails to load. This ensures your styles remain consistent.

Example: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Fallback Values in Variables</title>
    <style>
        .box {
            --primary-color: #3498db;
            background-color: var(--primary-color, #e74c3c); /* Fallback to red if primary-color is not defined */
            color: white;
            padding: 10px;
            margin: 10px;
        }

        .box-no-var {
            background-color: var(--non-existent-color, #2ecc71); /* Fallback to green */
            color: white;
            padding: 10px;
            margin: 10px;
        }
    </style>
</head>
<body>
    <div class="box">Box with Variable</div>
    <div class="box-no-var">Box with Fallback</div>
</body>
</html>

Example Code 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS Custom Properties Combined Example</title>
    <style>
        :root {
            --primary-color: #3498db;
            --secondary-color: #2ecc71;
            --background-color: #f0f0f0;
            --text-color: #333;
            --padding: 10px;
        }

        .dark-theme {
            --background-color: #333;
            --text-color: #f0f0f0;
        }

        body {
            background-color: var(--background-color);
            color: var(--text-color);
            font-family: Arial, sans-serif;
            padding: 20px;
            transition: background-color 0.5s, color 0.5s;
        }

        .box {
            background-color: var(--primary-color);
            padding: var(--padding);
            color: white;
            margin: 10px;
            transition: background-color 0.5s;
        }

        .box-secondary {
            background-color: var(--secondary-color);
        }

        .box-no-var {
            background-color: var(--non-existent-color, #2ecc71); /* Fallback to green */
            color: white;
            padding: var(--padding);
            margin: 10px;
        }
    </style>
</head>
<body>
    <button id="changeColor">Change Primary Color</button>
    <button id="toggleTheme">Toggle Theme</button>
    <div class="box">Primary Box</div>
    <div class="box box-secondary">Secondary Box</div>
    <div class="box-no-var">Box with Fallback</div>

    <script>
        document.getElementById('changeColor').addEventListener('click', function() {
            document.documentElement.style.setProperty('--primary-color', '#e74c3c');
        });

        document.getElementById('toggleTheme').addEventListener('click', function() {
            document.body.classList.toggle('dark-theme');
        });
    </script>
</body>
</html>
