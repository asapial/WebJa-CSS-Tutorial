<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Advanced Animations Example</title>
  <style>
    body {
      font-family: 'Arial', sans-serif;
      background: linear-gradient(135deg, #f06, #4a90e2);
      display: flex;
      justify-content: center;
      align-items: center;
      height: 100vh;
      margin: 0;
    }
    .container {
      background: #fff;
      padding: 20px;
      border-radius: 10px;
      box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2);
      text-align: center;
      width: 300px;
    }
    .box {
      width: 100px;
      height: 100px;
      margin: 20px auto;
      background: #4a90e2;
      border-radius: 10px;
      animation: exampleAnimation 2s linear 1 forwards;
    }
    @keyframes exampleAnimation {
      0% { transform: translateX(0); }
      100% { transform: translateX(200px); }
    }
    .controls {
      display: flex;

    .controls input::-webkit-outer-spin-button,
    .controls input::-webkit-inner-spin-button {
      -webkit-appearance: none;
      margin: 0;
    }
  </style>

<body>
  <div class="container">
    <div class="box" id="box"></div>
    <div class="controls">
      <label for="timingFunction">Timing Function:</label>
      <select id="timingFunction">
        <option value="linear">Linear</option>
        <option value="ease">Ease</option>
        <option value="ease-in">Ease-in</option>
        <option value="ease-out">Ease-out</option>
        <option value="ease-in-out">Ease-in-out</option>
      </select>
      <label for="delay">Delay (s):</label>
      <input type="number" id="delay" step="0.1" value="0">
      <label for="iterationCount">Iteration Count:</label>
      <input type="number" id="iterationCount" min="1" value="1">
      <label for="direction">Direction:</label>
      <select id="direction">
        <option value="normal">Normal</option>
        <option value="reverse">Reverse</option>
        <option value="alternate">Alternate</option>
        <option value="alternate-reverse">Alternate-reverse</option>
      </select>
      <label for="fillMode">Fill Mode:</label>
      <select id="fillMode">
        <option value="none">None</option>
        <option value="forwards">Forwards</option>
        <option value="backwards">Backwards</option>
        <option value="both">Both</option>
      </select>
    </div>
  </div>

  <script>
    const box = document.getElementById('box');
    const timingFunctionInput = document.getElementById('timingFunction');
    const delayInput = document.getElementById('delay');
    const iterationCountInput = document.getElementById('iterationCount');
    const directionInput = document.getElementById('direction');
    const fillModeInput = document.getElementById('fillMode');

    // Function to update the animation properties based on user input
    function updateAnimation() {
      const timingFunction = timingFunctionInput.value;
      const delay = delayInput.value + 's';
      const iterationCount = iterationCountInput.value;
      const direction = directionInput.value;
      const fillMode = fillModeInput.value;

      // Reset animation by removing and re-adding the animation property
      box.style.animation = 'none';
      box.offsetHeight; /* Trigger a reflow to restart the animation */
      box.style.animation = `exampleAnimation 2s ${timingFunction} ${delay} ${iterationCount} ${direction} ${fillMode}`;
    }

    // Add event listeners to update the animation whenever an input changes
    timingFunctionInput.addEventListener('change', updateAnimation);
    delayInput.addEventListener('input', updateAnimation);
    iterationCountInput.addEventListener('input', updateAnimation);
    directionInput.addEventListener('change', updateAnimation);
    fillModeInput.addEventListener('change', updateAnimation);

    // Initial call to set the animation with default values
    updateAnimation();
  </script>
</body>
</html>

HTML Structure

  • Document Structure:
    • The HTML document starts with a standard structure including <html>, <head>, and <body> tags.
    • <head> contains metadata like character set, viewport settings for responsiveness, and the document title.
  • Body Content:
    • The body contains a .container div that serves as a visual container for the animated box and controls.
    • Inside .container, there's a .box div which represents the element being animated.
    • Next to .box, there's a .controls div which contains various form elements (<select> and <input> tags) to adjust animation properties interactively.

CSS Styles

  • Global Styles:
    • body: Sets a background gradient (linear-gradient) spanning from top-left (#f06) to bottom-right (#4a90e2), providing a visually appealing backdrop.
    • .container: Styled with a white background, padding, border radius, and a subtle box shadow to create a visually distinct container for the animation demo.
    • .box: Represents the animated element (100px x 100px) with a rounded border (border-radius) and initial animation settings applied (exampleAnimation).
    • @keyframes exampleAnimation: Defines the animation keyframes where the .box element translates horizontally from 0% to 100% along the x-axis.

JavaScript Functionality

  • Animation Control:
    • JavaScript is used to dynamically update the animation properties based on user input.
    • Event listeners (addEventListener) are attached to the form elements (<select> and <input>):
      • timingFunctionInput: Changes the timing function of the animation.
      • delayInput: Adjusts the delay before the animation starts.
      • iterationCountInput: Sets how many times the animation repeats.
      • directionInput: Controls the direction of the animation (normal, reverse, alternate, alternate-reverse).
      • fillModeInput: Specifies how the animation styles are applied before and after it executes.
  • Update Animation Function:
    • updateAnimation(): A function that triggers whenever an input changes.
    • It reads the current values from the form elements, constructs a new animation string, and applies it to the .box element's style.
    • To ensure the animation restarts with new settings, it first resets the animation (box.style.animation = 'none';), triggers a reflow (box.offsetHeight;), and then reapplies the animation (box.style.animation = ...;).
