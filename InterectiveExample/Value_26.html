  background: $secondary-color;
  padding: 20px;
  border-radius: 8px;
  display: flex;
  align-items: center;
  justify-content: center;
  text-align: center;

.grid-container {
  display: grid;
  grid-gap: 20px;
  margin-top: 20px;

  .box {
    @extend %box-default;
  }

  .box-mixin {
    @include box-style;
  }
}

Explanation

SASS/SCSS Code
1. Introduction to SASS/SCSS
  • We use SCSS syntax, which is similar to CSS but with enhanced features.
2. Variables in SASS/SCSS
  • Variables $primary-color, $secondary-color, and $font-stack are defined for reusable values.
3. Nesting in SASS/SCSS
  • Nesting is used to define styles for .header and .footer within .main-container for better organization and readability.
4. Mixins in SASS/SCSS
  • Mixin @mixin box-style is created to encapsulate common styles for boxes and is included in elements using @include.
5. Inheritance in SASS/SCSS
  • Inheritance using %box-default is used to define default styles for boxes, and @extend is used to apply these styles.
HTML and Compiled CSS
  • The HTML structure uses the compiled CSS for styling.
  • Boxes demonstrate the use of mixins and inheritance with different class names (.box for inheritance and .box-mixin for mixin).
This example provides a comprehensive and interactive demonstration of SASS/SCSS features with a premium UI design.
